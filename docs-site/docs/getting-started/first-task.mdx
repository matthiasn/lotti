# Create your first task

A task is the home for an outcome, its checklist, and the notes, recordings,
images, or time records that explain the work.

## Create it

1. Open **Tasks**.
2. Use the **+** action and choose **Create task**.
3. Write an outcome-oriented title.
4. Choose a category when one clearly applies.
5. Save the task.

“Publish the V1 manual” is more useful than “Documentation” because it tells
you what finished looks like.

## Add useful context

Open the task and use its add action to attach a note, recording, image, or
other entry. Keep material with the task when it would help you—or an AI
assistant—understand the next action later.

## Make it actionable

Add a short checklist with concrete steps. If an AI provider and the relevant
skills are configured, Lotti can also turn a recording or written context into
checklist suggestions. Review proposed changes before applying them.

## Decide when it matters

- Set a due date when there is a real deadline.
- Set priority to express urgency.
- Bring the task into a Daily OS check-in when it belongs in today's plan.

Continue with [task management](../organize-and-reflect/tasks.mdx) for priority,
filtering, and voice-assisted workflows.
