## What Lotti keeps separate

- **Tasks** describe work and its status.
- **Plans** describe what you intend to do and when.
- **Entries** preserve notes, recordings, images, measurements, and other
  evidence.
- **Time records** describe what actually happened.
- **Categories** create boundaries for filtering, automation, and reflection.

That separation is deliberate: a plan can change without rewriting history,
and actual time can differ from the plan without pretending the day went
differently.

:::tip Start small

Create a couple of meaningful categories, one real task, and one Daily OS
check-in. Add more structure only when it earns its place.

:::
