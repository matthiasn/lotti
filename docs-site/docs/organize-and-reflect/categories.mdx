# Create useful categories

Categories represent areas you want to separate when planning or reviewing:
Health, Family, Research, a client, or a long-running responsibility. They can
color and filter tasks, entries, habits, and dashboards, and can control which
content Daily OS or an AI profile may use.

## Create a category

1. Open **Settings → Categories**.
2. Use the add action.
3. Choose a clear name and a color you can recognize quickly.
4. Configure planning or inference options only when the category needs them.
5. Save.

## Keep the category system small

Start with boundaries that change a decision. “Health” may be useful if you
filter or review it separately; “Miscellaneous” usually hides information. Add
categories later when a repeated need appears.

## Daily OS access

Enable **Day planning** only for categories whose tasks may enter the planning
ritual. This makes the planning context more intentional and avoids sending
unrelated category material to the selected inference endpoint.
