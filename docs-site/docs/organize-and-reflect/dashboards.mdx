# Build dashboards

A dashboard groups charts for measurements you want to interpret together.
Build one around a question—“How is training changing?”—instead of collecting
every available number on one screen.

## Create a dashboard

1. Open **Settings → Dashboard management**.
2. Add a dashboard with a clear name and optional description.
3. Add one or more measurable-data charts.
4. Reorder the charts so the most useful signal appears first.
5. Save, then open the dashboard from the Dashboards area.

Health data may be imported when a dashboard first requests it. Platform
permission prompts and available data types differ by device.
