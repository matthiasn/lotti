# Remember meaningful events

Events give meaningful moments their own place in Lotti. Use an event for a
launch, trip, celebration, conference, or other moment you want to revisit. Use
tasks for the preparation and follow-up work around it.

Events are currently an optional feature. Open **Settings → Advanced → Flags**
and turn on **Enable Events**. The **Events** destination then appears in the
desktop sidebar and under **More** on mobile.

## Browse and filter your events

The overview is photo-led so that an event is recognizable before you read its
title. The nearest future event appears in the **Upcoming** hero. Past events
are grouped by year, newest first.

Use **Search events** to find an event by its text. Select a category chip—such
as **Penguin Operations**, **Mission Control**, or **Fish Diplomacy**—to narrow
the complete archive. Filtering is applied to the stored event collection, not
only to the cards currently visible on screen.

Lotti loads a long archive in pages and renders only nearby rows, so hundreds
of photo-rich events stay practical without loading every cover at once.

## Create an event

Select **New event**. A new event opens on its detail page, where you can add a
title, category, date and time, status, cover photo, and supporting entries.

Choose a title that names the moment rather than its preparation. For example,
**Project Waddle launch gala** is the event; **Recalibrate the zero-gravity fish
feeder** is a linked task.

## Read and edit the event detail

The hero keeps the event's identity and main controls together:

- select the title to edit it;
- select the category or status pill to change it;
- select the date and time to reschedule the event;
- select a star to change the rating of a completed event; and
- open the overflow menu to change the cover or delete the event.

Changes save through the detail page; there is no separate form to reopen. A
rating appears after an event is completed, or whenever the event already has a
rating.

The summary uses the newest linked AI response when one exists. Otherwise it
shows the event's own note. Treat it as a quick recap, then use the photos and
timeline for the underlying record.

## Build a photographic record

The **Photos** section collects every image linked to the event in time order.
Select a thumbnail to open the full-screen viewer, then swipe between photos or
pinch to zoom. A large collection is capped to a compact preview; the last tile
shows how many more photos are available.

The cover is one of the event's linked photos. If no cover has been chosen,
Lotti can use the first linked photo. Use **Change cover** in the overflow menu
to choose another image, or add a new photo from the same picker.

## Follow the timeline and related work

/>

The timeline sorts linked entries from oldest to newest. It can contain:

- photos and their captions;
- notes;
- audio entries and voice notes; and
- time recordings, shown as a start, end, and elapsed span.

Select a timeline item to open its source entry. Select **Add** beside
**Timeline** to create a note, photo, audio entry, or other supported entry that
is linked to this event automatically.

The **Tasks** section is for preparation and follow-up. Select an existing task
to open it, or select **Add** to create a task already linked to the event. On
desktop, tasks remain visible in a right-hand rail while you read the event; on
mobile, they follow the timeline in the same scroll.

## Event or task?

Use this distinction when deciding where something belongs:

| Use an event for… | Use a task for… |
| --- | --- |
| A moment with a date, photos, recap, and memory value | An action with a status, estimate, checklist, or work session |
| Project Waddle launch gala | Recalibrate the zero-gravity fish feeder |
| Europa sardine futures summit | Confirm the interplanetary sardine cargo pods |

Link the two when the work supports the moment. This keeps the event enjoyable
to revisit while its operational work remains actionable.
