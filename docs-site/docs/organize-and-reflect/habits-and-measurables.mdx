# Habits & measurables

A habit records whether a recurring behavior succeeded, failed, or should be
skipped. A measurable records a numeric observation. They can support one
another without being the same thing: “morning exercises” is a habit, while
“pull-ups” and “minutes trained” are measurable values.

## Create a habit

1. Open **Settings → Habits**.
2. Add a habit and give it a specific title.
3. Assign a category when that helps filtering and recognition.
4. Configure its schedule and start date.
5. Save it.

When completing a habit, choose the state that preserves what happened:

- **Success** — you completed the intended behavior.
- **Fail** — the behavior was possible but did not happen.
- **Skip** — it was not applicable or not reasonably possible.

Do not use success merely to remove an uncomfortable red state; honest history
is more useful than a perfect chart.

## Create a measurable data type

Open **Settings → Measurable data types**, then add a name, optional description,
unit, and aggregation behavior. Aggregation determines whether a dashboard
shows individual values or combines observations over a day.
