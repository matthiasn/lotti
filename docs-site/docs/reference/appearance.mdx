# Choose Lotti's appearance

Open **Settings → Theming** to control how Lotti looks. The display mode and
the two palette choices are independent, so you can pair a bright daytime
theme with a completely different nighttime theme.

## Select a display mode

Choose one of three modes:

- **Dark** always uses the selected dark theme.
- **Automatic** follows the operating system's current light or dark setting.
- **Light** always uses the selected light theme.

Automatic mode is useful when your device changes appearance on a schedule.
Lotti follows that change while retaining your chosen palette for each side.

## Choose the light and dark palettes

1. Select **Light theme** or **Dark theme**.
2. Choose a named palette from the sheet.
3. Repeat for the other appearance if needed.

There is no Save button. Display mode and palette choices persist as soon as
you select them. The manual's own light and dark switch is separate: it changes
the documentation and automatically chooses the matching screenshot variant,
but it does not change your Lotti settings.
