# Work with keyboard shortcuts

Desktop Lotti exposes a searchable catalog of app-wide and contextual
commands. Open **Settings → Keyboard shortcuts** to see each command with the
binding resolved for your platform.

## Open shortcut help quickly

Press **F1** or **Primary + ?** to open the shortcut reference without leaving
your current work. **Primary** means Command on macOS and Control on Windows or
Linux.

Frequently useful app-wide commands include:

| Action | Shortcut |
| --- | --- |
| Open the command palette | Primary + K |
| Create a text entry | Primary + N |
| Create a task | Primary + T |
| Open Tasks, Daily OS, Projects, or Habits | Primary + 1 / 2 / 3 / 4 |
| Open Dashboards, Journal, Events, or Settings | Primary + 5 / 6 / 7 / 8 |

Individual pages add contextual commands such as save, refresh, search, or
create. Their availability can depend on what is currently focused or whether
there are unsaved changes.

Use the search field when you know either the action or part of its key
combination. The in-app catalog is the authority if a binding changes in a
later version of Lotti.
