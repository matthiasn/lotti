# Choose a recording style

Open **Settings → Recording Style** to choose the visualization shown while
Lotti records audio. This changes presentation only; both styles represent the
same recording process.

## Compare the two styles

- **Modern** shows an animated energy orb that reacts to the current level.
- **Analogue** shows a traditional VU meter with a moving needle.

Select either card to make it the active style. There is no separate Save or
Continue action.

## Preview with your voice

Enable **Try with your voice** to drive the preview from the microphone. Speak
at the distance you expect to use while recording and watch how the selected
visual responds. Turn the option off when you have finished comparing styles.

The preview helps you choose a visual language; it is not an input-gain or
recording-quality calibration tool. See [Record audio](../plan-and-capture/recordings.mdx)
for the actual recording workflow.
