# Find settings

Settings are grouped by the capability they control. Desktop uses a persistent
tree with the selected page beside it; mobile uses a focused list and opens one
settings page at a time.

<ManualScreenshot

Common destinations include:

- **Categories** for planning boundaries and **Labels** for cross-cutting tags.
- **Habits** and **Dashboards** for reflection structures.
- **AI settings** for providers, models, profiles, and skills.
- **Daily OS** for the planner's inference profile and greeting.
- **Sync** for connection and activity state.

The exact set can vary with platform capabilities and enabled features.
