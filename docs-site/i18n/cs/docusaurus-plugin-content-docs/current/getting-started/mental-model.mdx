## Stavební prvky, aniž bychom předstírali, že jsou všechny stejné

| Stavební prvek | K čemu slouží | Příklad |
| --- | --- | --- |
| Kategorie | Smysluplná oblast života nebo práce | Zdraví, rodina, výzkum |
| Úkol | Výsledek, který potřebuje posun | Připravit vydání V1 |
| Kontrolní seznam | Konkrétní kroky uvnitř úkolu | Ověřit poznámky k vydání |
| Záznam | Kontext nebo podklad | Poznámka, obrázek, zvuková nahrávka |
| Plánovaný blok | Navržený úsek dne | Kontrola vydání, 10:00–11:00 |
| Časový záznam | Co se skutečně stalo | Kontrola vydání, 10:18–11:07 |
| Návyk | Opakované chování, které vyhodnocuješ | Vzít si léky |
| Měřitelná veličina | Číselné pozorování | Klidová tepová frekvence |
| Přehled | Pohled na užitečná měření | Přehled tréninku |

## Agenti jsou hosté, ne spolumajitelé

Agent může číst souvislosti, které mu dáš, vytvořit si názor, shrnout
nepřehledný úkol a navrhnout užitečnou změnu. Jeho zpráva je názor s dohledatelným
původem, ne nový fakt ve tvé historii. Navržené změny úkolu, kontrolního
seznamu, stavu nebo data čekají, až je potvrdíš nebo zamítneš. Výjimkou je
pouze první název nebo jazyk jinak prázdného úkolu — pak znovu platí stejné
pravidlo lidského schválení.

Není to jen zdvořilost přilepená na prompt. Lotti drží osobní deník a pracovní
paměť agentů oddělené, takže agent nemůže zacházet s tvou historií jako s
poznámkovým blokem. AI můžeš používat, tvarovat ji, nebo ji nechat úplně být.

## První praktický oběh

1. Vytvoř kategorie, které skutečně používáš při rozhodování.
2. Zachycuj práci jako úkoly a podpůrné materiály nechávej připojené k nim.
3. Pomocí Daily OS rozhodni, co patří do konkrétního dne.
4. Zaznamenej skutečný čas a pozorování, aniž bys je nutil odpovídat plánu.
5. V návycích a přehledech hledej vzorce, ne jednotlivé dokonalé dny.

## Udržuj jasné vlastnictví

- Propojený úkol vlastní svůj název a kategorii.
- Blok Daily OS vlastní své umístění a délku.
- Časový záznam vlastní popis toho, co se stalo.
- AI navrhuje; ty návrhy kontroluješ a rozhoduješ.

:::tip[Nejde o bezchybnou časovou osu]

Když se plán změnil, nech ho změnit. Když se den vyvíjel bokem, zaznamenej, co
se opravdu stalo. Užitečný systém ti ukáže tvar života; nechce po tobě, abys
pro databázi předváděl produktivnější verzi sebe.

:::
