## Co Lotti udržuje oddělené

- **Úkoly** popisují práci a její stav.
- **Plány** popisují, co a kdy chceš udělat.
- **Záznamy** uchovávají poznámky, nahrávky, obrázky, měření a další
  podklady.
- **Časové záznamy** popisují, co se skutečně stalo.
- **Kategorie** vytvářejí hranice pro filtrování, automatizaci a ohlédnutí.

Toto oddělení je záměrné: plán se může změnit bez přepisování historie a
skutečný čas se může od plánu lišit, aniž bys musel předstírat, že den proběhl
jinak.

:::tip Začni jednoduše

Vytvoř několik smysluplných kategorií, jeden skutečný úkol a jeden přehled v
Daily OS. Další strukturu přidávej teprve tehdy, když si své místo zaslouží.

:::
