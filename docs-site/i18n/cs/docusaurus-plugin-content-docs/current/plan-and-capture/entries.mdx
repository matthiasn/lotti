# Přidávání záznamů a kontextu

Lotti používá jednu nabídku **Přidat** pro věci, které během dne zachytíš.
Otevři ji z deníku pro samostatný záznam nebo z úkolu, aby nový záznam zůstal
k tomuto úkolu připojený.

## Vyber záznam odpovídající materiálu

| Akce | K čemu ji použít |
| --- | --- |
| **Kontrolní seznam** | Krátká sada konkrétních kroků uvnitř aktuálního úkolu |
| **Úkol** | Samostatný výsledek, který potřebuje vlastní stav a kontext |
| **Zvuková nahrávka** | Hlasová poznámka, kterou lze přehrát nebo přepsat |
| **Časovač** | Měřené pracovní sezení propojené s aktuálním kontextem |
| **Textový záznam** | Poznámky, rozhodnutí, pozorování nebo vložené informace |
| **Importovat obrázek** | Existující obrázek ze zařízení |
| **Snímek obrazovky** | Nový snímek plochy připojený jako obrázkový záznam |

Nabídka je kontextová. **Kontrolní seznam** je dostupný, když může úkol vlastnit
jeho položky; akce závislé na zařízení, například **Snímek obrazovky**, se
zobrazí jen na podporovaných platformách.

## Udržuj kontext u práce

Když otevřeš **Přidat** z úkolu, Lotti nový záznam s úkolem propojí. Nahrávky,
poznámky, obrázky a časové záznamy tak zůstanou pohromadě na stránce
podrobností úkolu. Samostatný záznam v deníku použij pro materiál, který ještě
nepatří ke konkrétnímu výsledku; propojit ho můžeš později.

Přečti si [Nahrávání zvuku](./recordings.mdx) o zachycení a přepisu nebo
[Práci s úkoly](../organize-and-reflect/tasks.mdx) o struktuře úkolu, titulních
obrázcích a propojeních.
