# Volba vzhledu Lotti

Vzhled Lotti řídíš v **Nastavení → Motivy**. Režim zobrazení a dvě volby palet
jsou nezávislé, takže můžeš spojit světlý denní motiv se zcela jiným nočním.

## Vyber režim zobrazení

Zvol jeden ze tří režimů:

- **Tmavý** vždy používá vybraný tmavý motiv.
- **Automatický** následuje aktuální světlé či tmavé nastavení operačního
  systému.
- **Světlý** vždy používá vybraný světlý motiv.

Automatický režim se hodí, když zařízení mění vzhled podle rozvrhu. Lotti změnu
následuje a zachovává vybranou paletu pro každou stranu.

## Vyber světlou a tmavou paletu

1. Vyber **Světlý motiv** nebo **Tmavý motiv**.
2. V panelu zvol pojmenovanou paletu.
3. Podle potřeby zopakuj pro druhý vzhled.

Tlačítko Uložit zde není. Režim a palety se uloží při výběru. Vlastní přepínač
světlého a tmavého režimu příručky je oddělený: mění dokumentaci a automaticky
vybírá odpovídající variantu snímku, ale nemění tvoje nastavení Lotti.
