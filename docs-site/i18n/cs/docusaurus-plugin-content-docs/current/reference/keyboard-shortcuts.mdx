# Práce s klávesovými zkratkami

Lotti na počítači nabízí prohledávatelný katalog příkazů platných v celé
aplikaci i v aktuálním kontextu. Otevři **Nastavení → Klávesové zkratky** a
zobraz každý příkaz s kombinací odpovídající platformě.

## Rychle otevři nápovědu zkratek

Stisknutím **F1** nebo **Primární + ?** otevři přehled bez opuštění aktuální
práce. **Primární** znamená Command v macOS a Control ve Windows nebo Linuxu.

Často užitečné globální příkazy:

| Akce | Zkratka |
| --- | --- |
| Otevřít paletu příkazů | Primární + K |
| Vytvořit textový záznam | Primární + N |
| Vytvořit úkol | Primární + T |
| Otevřít Úkoly, Daily OS, Projekty nebo Návyky | Primární + 1 / 2 / 3 / 4 |
| Otevřít Nástěnky, Deník, Události nebo Nastavení | Primární + 5 / 6 / 7 / 8 |

Jednotlivé stránky přidávají kontextové příkazy jako uložení, obnovení,
vyhledávání nebo vytvoření. Dostupnost může záviset na aktuálním zaměření nebo
neuložených změnách.

## Spusť příkaz bez opuštění práce

Stisknutím **Primární + K** otevři paletu příkazů nad aktuální stránkou.
Kombinuje globální akce — vytváření záznamů, otevírání úkolů, navigaci a
přiblížení — s příkazy dodanými aktivní stránkou.

Psaním zuž seznam podle názvu akce, skupiny nebo kombinace. Šipkami nahoru a
dolů pohybuj zvýrazněním, klávesou **Enter** příkaz spusť a **Escape** paletu
zavři. Protože je paleta kontextová, příkaz **Uložit** se zobrazí jen tehdy,
když zaměřený editor může právě ukládat.

aplikaci je autoritou, pokud se vazba v pozdější verzi Lotti změní.
