# Volba stylu nahrávání

V **Nastavení → Styl nahrávání** vyber vizualizaci zobrazovanou při nahrávání
zvuku v Lotti. Mění se jen prezentace; oba styly představují stejný proces.

## Porovnej dva styly

- **Moderní** ukazuje animovanou energetickou kouli reagující na úroveň.
- **Analogový** ukazuje tradiční VU měřič s pohyblivou ručičkou.

Výběrem karty nastav aktivní styl. Není zde samostatná akce Uložit ani
Pokračovat.

## Vyzkoušej svůj hlas

Zapni **Vyzkoušet s hlasem** a nech mikrofon řídit náhled. Mluv ze
vzdálenosti, kterou očekáváš při nahrávání, a sleduj reakci vizualizace. Po
porovnání stylů možnost vypni.

Náhled pomáhá vybrat vizuální jazyk; není nástrojem kalibrace vstupního
zesílení ani kvality nahrávky. Skutečný postup popisuje
[Nahrávání zvuku](../plan-and-capture/recordings.mdx).
