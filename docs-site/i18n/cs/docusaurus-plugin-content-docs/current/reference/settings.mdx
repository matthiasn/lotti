# Hledání nastavení

Nastavení jsou seskupená podle schopnosti, kterou řídí. Počítač používá trvalý
strom s vybranou stránkou vedle něj; mobil používá soustředěný seznam a otevírá
vždy jednu stránku nastavení.

V samostatné poslední části Nastavení zvol **Příručka**, abys tuto příručku
otevřel ve výchozím prohlížeči. V macOS vede na stejné místo také **Nápověda →
Příručka**. Příručka se ve výchozím nastavení řídí jazykem systému; jiný jazyk
vybereš v **Nastavení → Pokročilé nastavení → Jazyk**.

Časté cíle zahrnují:

- **[Vzhled](./appearance.mdx)** pro světlý, tmavý a automatický režim a
  nezávislé palety motivů.
- **[Styl nahrávání](./recording-style.mdx)** pro energetickou kouli nebo
  analogový měřič při nahrávání.
- **[Řeč](./speech.mdx)** pro místní hlas, model a rychlost čtení shrnutí úkolů.
- **[Klávesové zkratky](./keyboard-shortcuts.mdx)** pro prohledávatelný katalog
  příkazů na počítači.
- **[Oslavy dokončení](./completion-celebrations.mdx)** pro animace, haptiku,
  styly a jejich živé zkušební prostředí.
- **[Pokročilé nastavení](./advanced-settings.mdx)** pro přepínače funkcí,
  protokolování, údržbu, diagnostiku průvodce a informace o sestavení.
- **Kategorie** pro hranice plánování a **Štítky** pro značky napříč oblastmi.
- **Návyky** a **Panely** pro struktury ohlédnutí.
- **Nastavení AI** pro poskytovatele, modely, profily a dovednosti.
- **Daily OS** pro profil inference plánovače a oslovení.
- **Sync** pro stav připojení a aktivity.

Přesná sada se může lišit podle schopností platformy a zapnutých funkcí.

## Spravuj definice

Struktury organizující data spravuj v **Nastavení → Definice**. Kategorie
určují hranice plánování a soukromí, štítky označují úkoly napříč kategoriemi,
návyky definují opakované rutiny, panely skládají pohledy ohlédnutí a
měřitelné veličiny definují čísla sledovaná v čase.
