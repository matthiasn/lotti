# Volba hlasu pro čtení

Když je hlasité čtení dostupné, otevři **Nastavení → Řeč**. Ovládání platí pro
AI shrnutí úkolů; nemění zachycení mikrofonu ani poskytovatele AI, který
přepisuje nahrávku.

## Vyber hlas

Pomocí přepínače **Ženský / Mužský** procházej dvě skupiny a potom vyber jeden
z pěti číslovaných hlasů. Přepnutí skupiny mění jen viditelný seznam; aktivní
hlas nastavíš výběrem konkrétního čísla.

Vybraný hlas je místní předvolba zařízení. Každé zařízení tak může používat
hlas vhodný pro své reproduktory a prostředí poslechu, aniž by změnilo ostatní
zařízení.

## Stažení modelu a soukromí

Lotti nyní používá pro hlasité čtení **Supertonic 3**. Syntéza běží na zařízení
a neposílá shrnutí cloudové hlasové službě. Model je velký a stáhne se při
prvním použití, proto pro úvodní nastavení počítej s časem, sítí a úložištěm.

## Nastav rychlost čtení

Vyber rychlost od **0,5×** do **2×**. Hodnota **1×** je přirozené tempo, nižší
hodnota briefing zpomalí a vyšší ho zrychlí. Rychlost se také ukládá jen na
aktuálním zařízení.

Tlačítko hlasitého čtení se zobrazí, jen když je funkce zapnutá, platforma
podporuje hlasový modul a úkol má neprázdné AI shrnutí.
