# Import zdravotních dat

Import zdraví je dostupný v iOS a Androidu. Otevři **Nastavení → Pokročilé →
Import zdraví**, vyber počáteční a koncové datum a potom rodinu dat, kterou má
Lotti přečíst z úložiště zdravotních údajů telefonu.

Dostupné importy jsou:

- **Aktivita** pro souhrny aktivity;
- **Spánek** pro vzorky spánku;
- **Tepová frekvence** pro pozorování pulzu;
- **Krevní tlak** pro jeho hodnoty;
- **Tělesné měření** pro tělesné metriky; a
- **Trénink** pro zaznamenané tréninky.

Každé tlačítko importuje jen vlastní rodinu dat v zobrazeném rozsahu. Samotná
změna rozsahu import nespustí.

Operační systém může požádat o oprávnění ke zdravotním datům. Lotti může číst
jen typy, které povolíš. Pokud očekávaná rodina chybí, zkontroluj oprávnění
aplikace v systémovém nastavení telefonu a import zopakuj.

Import zdraví se na počítači nezobrazuje, protože tam základní mobilní úložiště
nejsou dostupná. Importovaná pozorování lze po synchronizaci na další zařízení
stále prohlížet v odpovídajících přehledech Lotti.
