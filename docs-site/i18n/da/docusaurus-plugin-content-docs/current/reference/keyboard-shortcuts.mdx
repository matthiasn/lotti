# Arbejd med tastaturgenveje

Desktop Lotti eksponerer et søgbart katalog over app-dækkende og kontekstuelle
kommandoer. Åbn **Indstillinger → genveje** for at se hver kommando med
Binding løst for din platform.

## Hjælp hurtigt med en genvej

Tryk **F1** eller **Primær + ?** for at åbne genvejsreferencen uden at forlade
dit nuværende arbejde. **Primær** betyder Command på macOS og Control på Windows eller
Linux.

Ofte nyttige app-omfattende kommandoer inkluderer:

| Handling | Genvej |
| --- | --- |
| Åbn kommandopaletten | Primary + K |
| Opret en tekstindtastning | Primær + N |
| Opret en opgave | Primær + T |
| Åbne opgaver, dagligt styresystem, projekter eller vaner | primært + 1 / 2 / 3 / 4 |
| Åbne dashboards, journal, begivenheder eller indstillinger | Primær + 5 / 6 / 7 / 8 |

Individuelle sider tilføjer kontekstuelle kommandoer som gem, opdater, søg eller
skab. Deres tilgængelighed kan afhænge af, hvad der aktuelt er fokuseret, eller om
Der er ændringer uden gemme.

## Kør en kommando uden at forlade dit arbejde

Tryk **Primær + K** for at åbne kommandopaletten over den aktuelle side. Det
kombinerer app-omfattende handlinger – oprettelse af indgange, åbning af opgaver, navigation og
Zooming—med kommandoer leveret af den aktive side.

<ManualScreenshot
  caseId="keyboard-shortcuts/palette"

/>

Begynd at skrive for at indsnævre listen efter handlingsnavn, gruppe eller tastkombination. Anvendelse
op- og nedpilene for at flytte markeringen, **Enter** for at køre det, og
**Escape** for at lukke paletten. Fordi paletten er kontekstuel, en kommando
såsom **Gem** vises kun, når den fokuserede editor aktuelt kan gemme.

Brug søgefeltet, når du kender enten handlingen eller en del af dens nøgle
kombination. Kataloget i appen er autoriteten, hvis en binding ændres i en
senere version af Lotti.
