# Find indstillinger

Indstillinger er grupperet efter den kapacitet, de kontrollerer. Skrivebordet bruger en persistent
træ med den valgte side ved siden af; Mobil bruger en fokuseret liste og åbner en
Indstillingsside ad gangen.

Vælg **Manual** i den sidste sektion af Indstillinger for at åbne denne guide
i din standardbrowser. På macOS giver **Hjælp → Manual** det samme link.
Guiden følger som standard dit systemsprog; Vælg et andet sprog
i **Indstillinger → Avancerede indstillinger → Sprog** efter behov.

Almindelige destinationer inkluderer:

- **[Appearance](./appearance.mdx)** for light, dark, and automatic display
  Flere uafhængige temapaletter.
- **[Recording Style](./recording-style.mdx)** for the energy-orb or analogue
  Måler vist under optagelsen.
- **[Speech](./speech.mdx)** for the on-device voice, model, and speed used to
  Læs opgaveoversigter højt.
- **[Keyboard shortcuts](./keyboard-shortcuts.mdx)** for the searchable desktop
  Kommandokatalog.
- **[Completion celebrations](./completion-celebrations.mdx)** for animations,
  haptik, stilarter og deres levende legeplads.
- **[Advanced Settings](./advanced-settings.mdx)** for feature flags, logging,
  vedligeholdelseshandlinger, onboarding-diagnostik og byggeinformation.
- **Kategorier** til planlægningsgrænser og **Etiketter** til krydsskæring af tags.
- **Habits** og **Dashboards** til refleksionsstrukturer.
- **AI-indstillinger** for udbydere, modeller, profiler og færdigheder.
- **Daily OS** for planlæggerens slutningsprofil og hilsen.
- ** Synkroniser** for forbindelse og aktivitetstilstand.

Det præcise sæt kan variere afhængigt af platformens kapaciteter og aktiverede funktioner.

## Administrér definitioner

Åbn **Indstillinger → Definitioner** for at administrere de strukturer, der organiserer din
Data. Kategorier sætter grænser for planlægning og privatliv, labels tagger opgaver på tværs
Kategorier, vaner definerer tilbagevendende rutiner, dashboards arrangerer refleksion
visninger og målbare tal definerer de tal, du følger over tid.
