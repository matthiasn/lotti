# Deine erste Aufgabe erstellen

Eine Aufgabe ist das Zuhause für ein Ergebnis, seine Checkliste und alle
Notizen, Aufnahmen, Bilder oder Zeiteinträge, die die Arbeit erklären.

## Mit dem schnellsten Weg beginnen

Wenn du noch im Onboarding bist, sprich das Ergebnis aus oder wähle **Lieber
tippen?**. Bestimme den Bereich, zu dem die Aufgabe gehört. Lotti nutzt diese
Kategorie für die richtigen Ansichten und die konfigurierte Automatisierung.

echte Aufgabentitel. Öffne die Karte, um im normalen Aufgabenarbeitsbereich
weiterzumachen.

## Eine Aufgabe aus der Aufgabenliste erstellen

1. Öffne **Aufgaben**.
2. Wähle **+** und **Aufgabe**.
3. Formuliere einen ergebnisorientierten Titel.
4. Wähle eine Kategorie, wenn sie eindeutig passt.
5. Speichere.

Auf dem Desktop zeigt der Arbeitsbereich Liste und ausgewählte Aufgabe
gemeinsam. Mobil öffnet die Auswahl die Details im Vollbild. In beiden
Ansichten machen Titelgrafiken und Kategorien dichte Listen leichter lesbar.

„V1-Handbuch veröffentlichen“ ist nützlicher als „Dokumentation“, weil der
Titel sagt, wie „fertig“ aussieht.

## Nützlichen Kontext hinzufügen

Öffne die Aufgabe und hänge über **Hinzufügen** eine Notiz, Aufnahme, ein Bild
oder einen anderen Eintrag an. Behalte Material bei der Aufgabe, wenn es dir –
oder einem KI-Assistenten – später beim nächsten Schritt hilft.

## Die Aufgabe umsetzbar machen

Füge eine kurze Checkliste mit konkreten Schritten hinzu. Sind KI-Anbieter und
passende Fähigkeiten konfiguriert, kann Lotti auch aus Aufnahme oder Text
Checklisten-Vorschläge erstellen. Prüfe Änderungen vor dem Anwenden.

## Entscheiden, wann sie wichtig ist

- Setze ein Fälligkeitsdatum nur bei einer echten Frist.
- Verwende die Priorität für Dringlichkeit.
- Nimm die Aufgabe in einen Daily-OS-Check-in auf, wenn sie in den heutigen
  Plan gehört.
