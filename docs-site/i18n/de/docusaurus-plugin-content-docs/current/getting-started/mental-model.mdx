## Die Bausteine, ohne so zu tun, als wären sie alle dasselbe

| Baustein | Dafür verwendest du ihn | Beispiel |
| --- | --- | --- |
| Kategorie | Ein sinnvoller Lebens- oder Arbeitsbereich | Gesundheit, Familie, Forschung |
| Aufgabe | Ein Ergebnis, das Fortschritt braucht | V1-Release vorbereiten |
| Checkliste | Konkrete Schritte innerhalb einer Aufgabe | Versionshinweise prüfen |
| Eintrag | Kontext oder Beleg | Notiz, Bild, Audioaufnahme |
| Geplanter Block | Ein vorgesehener Zeitraum im Tag | Release-Prüfung, 10:00–11:00 Uhr |
| Zeiteintrag | Was tatsächlich passiert ist | Release-Prüfung, 10:18–11:07 Uhr |
| Gewohnheit | Ein wiederkehrendes Verhalten, das du bewertest | Medikamente einnehmen |
| Messgröße | Eine numerische Beobachtung | Ruhepuls |
| Dashboard | Eine Ansicht nützlicher Messwerte | Trainingsübersicht |

## Agenten sind Gäste, keine Miteigentümer

Ein Agent kann den Kontext lesen, den du ihm gibst, sich eine Meinung bilden,
eine unübersichtliche Aufgabe zusammenfassen und eine sinnvolle Änderung
vorschlagen. Er darf deine Aufzeichnung nicht heimlich ändern. Vorgeschlagene
Änderungen an Aufgabe, Checkliste, Status oder Datum warten darauf, dass du sie
bestätigst oder verwirfst. Die engen Ausnahmen sind ein erster Titel oder eine
erste Sprache für eine sonst leere Aufgabe – danach gilt dieselbe Regel der
menschlichen Freigabe.

Das ist nicht bloß Höflichkeit im Prompt. Lotti hält persönliches Logbuch und
Arbeitsgedächtnis der Agenten getrennt, sodass der Agent deine Geschichte nicht
als Notizzettel behandeln kann. Du kannst KI einsetzen, sie formen oder sie
komplett in Ruhe lassen.

## Ein erster praktischer Umlauf

1. Erstelle Kategorien, die du wirklich für Entscheidungen verwendest.
2. Halte Arbeit als Aufgaben fest und hänge unterstützendes Material dort an.
3. Entscheide mit Daily OS, was in einen bestimmten Tag gehört.
4. Erfasse tatsächliche Zeit und Beobachtungen, ohne sie in den Plan zu zwingen.
5. Suche in Gewohnheiten und Dashboards nach Mustern statt nach einzelnen
   perfekten Tagen.

## Halte die Zuständigkeit klar

- Eine verknüpfte Aufgabe besitzt ihren Titel und ihre Kategorie.
- Ein Daily-OS-Block besitzt seine Position und Dauer.
- Ein Zeiteintrag besitzt die Aufzeichnung dessen, was passiert ist.
- Die KI schlägt vor; du prüfst und entscheidest.

:::tip Es geht nicht um einen makellosen Zeitstrahl

Wenn sich der Plan geändert hat, darf er sich ändern. Wenn der Tag seitwärts
lief, halte fest, was tatsächlich geschah. Ein nützliches System zeigt dir die
Form deines Lebens; es verlangt nicht, dass du für die Datenbank ein
produktiveres aufführst.

:::
