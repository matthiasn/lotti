## Was Lotti bewusst trennt

- **Aufgaben** beschreiben Arbeit und ihren Status.
- **Pläne** beschreiben, was du wann tun möchtest.
- **Einträge** bewahren Notizen, Aufnahmen, Bilder, Messwerte und andere Belege.
- **Zeiteinträge** beschreiben, was tatsächlich passiert ist.
- **Kategorien** schaffen Grenzen für Filter, Automatisierung und Reflexion.

Diese Trennung ist Absicht: Ein Plan darf sich ändern, ohne die Vergangenheit
umzuschreiben. Und die tatsächliche Zeit darf vom Plan abweichen, ohne so zu
tun, als wäre der Tag anders verlaufen.

:::tip Fang klein an

Erstelle ein paar sinnvolle Kategorien, eine echte Aufgabe und einen Daily-OS-
Check-in. Füge erst dann mehr Struktur hinzu, wenn sie dir wirklich hilft.

:::
