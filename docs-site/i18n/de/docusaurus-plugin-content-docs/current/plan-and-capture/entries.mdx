# Einträge und Kontext hinzufügen

Lotti verwendet ein gemeinsames **Hinzufügen**-Menü für alles, was du im
Tagesverlauf festhältst. Öffne es im Journal für einen eigenständigen Eintrag
oder innerhalb einer Aufgabe, damit der neue Eintrag dort verknüpft bleibt.

## Den passenden Eintrag wählen

| Aktion | Dafür verwendest du sie |
| --- | --- |
| **Checkliste** | Eine kurze Folge konkreter Schritte in der aktuellen Aufgabe |
| **Aufgabe** | Ein separates Ergebnis mit eigenem Status und Kontext |
| **Audioaufnahme** | Eine Sprachnotiz zum Abspielen oder Transkribieren |
| **Timer** | Eine zeitlich erfasste Arbeitssitzung im aktuellen Kontext |
| **Texteintrag** | Notizen, Entscheidungen, Beobachtungen oder eingefügte Informationen |
| **Bild importieren** | Ein vorhandenes Bild vom Gerät |
| **Screenshot** | Ein neuer Desktop-Screenshot als Bildeintrag |

Das Menü berücksichtigt den Kontext. **Checkliste** erscheint, wenn eine
Aufgabe die Einträge besitzen kann. Gerätespezifische Aktionen wie
**Screenshot** erscheinen nur auf unterstützten Plattformen.

## Kontext bei der Arbeit behalten

Öffnest du **Hinzufügen** innerhalb einer Aufgabe, verknüpft Lotti den neuen
Eintrag mit ihr. Aufnahmen, Notizen, Bilder und Zeiteinträge bleiben dadurch
gemeinsam auf der Aufgabendetailseite. Verwende einen eigenständigen
Journaleintrag, wenn Material noch keinem konkreten Ergebnis gehört; du kannst
es später verknüpfen.

Unter [Audio aufnehmen](./recordings.mdx) findest du Aufnahme und Transkription.
[Mit Aufgaben arbeiten](../organize-and-reflect/tasks.mdx) erklärt Struktur,
Titelgrafiken und Verknüpfungen.
