# Lottis Erscheinungsbild wählen

Öffne **Einstellungen → Design**, um Lottis Aussehen zu steuern. Anzeigemodus
und die beiden Paletten sind unabhängig. So kannst du tagsüber ein helles und
nachts ein völlig anderes dunkles Design verwenden.

## Einen Anzeigemodus wählen

Wähle einen von drei Modi:

- **Dunkel** verwendet immer das gewählte dunkle Design.
- **Automatisch** folgt der aktuellen Hell-/Dunkel-Einstellung des Betriebssystems.
- **Hell** verwendet immer das gewählte helle Design.

Der automatische Modus ist praktisch, wenn dein Gerät sein Erscheinungsbild
nach Zeitplan wechselt. Lotti folgt diesem Wechsel und behält auf beiden Seiten
die jeweils gewählte Palette bei.

## Helle und dunkle Palette wählen

1. Wähle **Helles Design** oder **Dunkles Design**.
2. Wähle im Dialog eine benannte Palette.
3. Wiederhole das bei Bedarf für den anderen Modus.

Eine Schaltfläche zum Speichern gibt es nicht. Modus und Palette werden direkt
nach der Auswahl gespeichert. Der Hell-/Dunkel-Schalter dieses Handbuchs ist
davon unabhängig: Er ändert die Dokumentation und wählt automatisch das
passende Screenshot-Motiv, aber nicht deine Lotti-Einstellungen.
