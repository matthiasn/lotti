# Mit Tastenkürzeln arbeiten

Lotti zeigt auf dem Desktop einen durchsuchbaren Katalog globaler und
kontextbezogener Befehle. Öffne **Einstellungen → Tastenkürzel**, um jeden
Befehl mit der Belegung deiner Plattform zu sehen.

## Die Hilfe schnell öffnen

Drücke **F1** oder **Primär + ?**, um die Kürzelübersicht über deiner aktuellen
Arbeit zu öffnen. **Primär** bedeutet Befehl auf macOS und Strg unter Windows
oder Linux.

Häufig nützliche globale Befehle:

| Aktion | Kürzel |
| --- | --- |
| Befehlspalette öffnen | Primär + K |
| Texteintrag erstellen | Primär + N |
| Aufgabe erstellen | Primär + T |
| Aufgaben, Daily OS, Projekte oder Gewohnheiten öffnen | Primär + 1 / 2 / 3 / 4 |
| Dashboards, Journal, Ereignisse oder Einstellungen öffnen | Primär + 5 / 6 / 7 / 8 |

Einzelne Seiten ergänzen Befehle wie Speichern, Aktualisieren, Suchen oder
Erstellen. Ihre Verfügbarkeit kann vom Fokus oder ungespeicherten Änderungen
abhängen.

## Einen Befehl ohne Unterbrechung ausführen

Drücke **Primär + K**, um die Befehlspalette über der aktuellen Seite zu
öffnen. Sie kombiniert globale Aktionen – Einträge erstellen, Aufgaben öffnen,
navigieren und zoomen – mit Befehlen der aktiven Seite.

Tippe, um nach Aktionsname, Gruppe oder Tastenkombination zu filtern. Bewege die
Markierung mit Pfeil hoch und runter, führe mit **Eingabe** aus und schließe mit
**Escape**. Weil die Palette kontextbezogen ist, erscheint etwa **Speichern**
nur, wenn der fokussierte Editor gerade speichern kann.

## Nach Aktion oder Taste suchen

<ManualScreenshot
  caseId="keyboard-shortcuts/search"
  alt="Lottis nach einem Suchbegriff gefilterter Tastenkürzel-Katalog"
  caption="Durchsuche den Katalog, statt ihn auswendig zu lernen. Die Treffer aktualisieren sich beim Tippen."
/>

Verwende das Suchfeld, wenn du entweder die Aktion oder einen Teil ihrer
Tastenkombination kennst. Falls sich eine Belegung in einer späteren
Lotti-Version ändert, ist der In-App-Katalog maßgeblich.
