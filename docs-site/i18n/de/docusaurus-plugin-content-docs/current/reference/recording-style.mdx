# Einen Aufnahmestil wählen

Öffne **Einstellungen → Aufnahmestil**, um die Darstellung während einer
Audioaufnahme zu wählen. Nur die Optik ändert sich; beide Stile zeigen
denselben Aufnahmevorgang.

## Die beiden Stile vergleichen

- **Modern** zeigt eine animierte Energiekugel, die auf den aktuellen Pegel reagiert.
- **Analog** zeigt ein klassisches VU-Meter mit beweglicher Nadel.

Wähle eine der Karten, um den Stil zu aktivieren. Eine zusätzliche Schaltfläche
zum Speichern oder Fortfahren gibt es nicht.

## Mit deiner Stimme ausprobieren

Aktiviere **Mit deiner Stimme ausprobieren**, damit das Mikrofon die Vorschau
steuert. Sprich aus der Entfernung, die du auch bei Aufnahmen verwenden wirst,
und beobachte die gewählte Darstellung. Schalte die Option nach dem Vergleich
wieder aus.

Die Vorschau hilft dir bei der Wahl der Optik; sie kalibriert weder die
Eingangsverstärkung noch die Aufnahmequalität. Den tatsächlichen Ablauf findest
du unter [Audio aufnehmen](../plan-and-capture/recordings.mdx).
