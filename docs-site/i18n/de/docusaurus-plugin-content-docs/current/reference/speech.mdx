# Vorlesestimme wählen

Wenn Vorlesen verfügbar ist, öffne **Einstellungen → Sprachausgabe**. Diese
Optionen gelten für KI-Zusammenfassungen von Aufgaben. Sie ändern weder die
Mikrofonaufnahme noch den KI-Anbieter für Transkriptionen.

## Eine Stimme wählen

Wechsle mit **Weiblich / Männlich** zwischen beiden Gruppen und wähle dann eine
der fünf nummerierten Stimmen. Der Gruppenwechsel zeigt zunächst nur die andere
Liste; erst die Wahl einer Nummer aktiviert die Stimme.

  caseId="speech/voice-options"

  caption="Zehn gebündelte Stimmstile sind verfügbar: Weiblich 1–5 und Männlich 1–5."
/>

Die gewählte Stimme ist eine lokale Geräteeinstellung. So kann jedes Gerät eine
Stimme verwenden, die zu Lautsprechern und Umgebung passt, ohne deine anderen
Geräte zu verändern.

## Modelldownload und Datenschutz

Lotti verwendet derzeit **Supertonic 3** zum Vorlesen. Die Synthese läuft auf
dem Gerät; die Zusammenfassung wird nicht an einen Cloud-Sprachdienst gesendet.
Das große Modell wird beim ersten Bedarf heruntergeladen. Plane dafür Zeit,
Netzwerkvolumen und Speicherplatz ein.

## Lesetempo einstellen

Wähle ein Tempo von **0,5×** bis **2×**. **1×** entspricht dem natürlichen
Tempo, niedrigere Werte verlangsamen und höhere verkürzen das Briefing. Auch
diese Auswahl bleibt auf dem aktuellen Gerät.

Die Vorleseschaltfläche erscheint nur, wenn das Feature aktiviert ist, die
Plattform die Sprachengine unterstützt und die Aufgabe eine nicht leere
KI-Zusammenfassung besitzt.
