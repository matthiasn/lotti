# Etiqueta el trabajo

Las etiquetas añaden una segunda capa flexible de organización. Una tarea tiene una categoría principal, pero puede llevar varias etiquetas para iniciativas, urgencia, estados de espera u otra propiedad que atraviese los límites de las categorías.

El espacio de trabajo Intergalactic Penguin Logistics usa las etiquetas de forma intencionada:

- **Project Waddle** conecta el trabajo de lanzamiento entre Operaciones de pingüinos, Control de misión y Diplomacia de los peces.
- **Hábitat crítico** identifica el trabajo que debe terminar antes de que la expedición entre en el hábitat orbital.
- **Esperando a Control de misión** registra un estado de bloqueo real.
- **Sensible al mercado de sardinas** mantiene identificable el trabajo privado de negociación.

## Crea y edita una etiqueta

1. Abre **Ajustes → Etiquetas**.
2. Selecciona **Crear etiqueta**.
3. Añade un nombre conciso y una descripción opcional.
4. Elige un color que siga distinguiéndose junto a otras etiquetas.
5. Decide si la etiqueta es privada y qué categorías pueden usarla.
6. Guarda.

Merece la pena escribir descripciones cuando una etiqueta pueda interpretarse de más de una forma. Aportan el significado operativo sin obligarte a repetir esa explicación en cada título de tarea.

Las **Categorías aplicables** limitan dónde se puede seleccionar la etiqueta. Project Waddle está disponible en las tres partes de la expedición implicadas en el lanzamiento, mientras que una etiqueta especializada puede quedar limitada a una categoría. Así los selectores siguen siendo relevantes en vez de presentar todas las etiquetas en todas partes.

## Aplica etiquetas a las tareas

Abre una tarea y usa su control de etiquetas para añadirlas o quitarlas. Las etiquetas funcionan junto con los filtros de estado, prioridad y categoría de la tarea, por lo que puedes plantear preguntas concretas como «mostrar trabajo abierto de P1 crítico para el hábitat» sin inventar una categoría nueva para esa vista temporal.

Quitar una etiqueta de una tarea se trata como un rechazo intencional cuando están activadas las sugerencias automáticas de etiquetas. Lotti registra esa decisión para que la misma sugerencia no se vuelva a aplicar de inmediato sin un motivo nuevo.

## Mantén las etiquetas útiles

- Prefiere etiquetas que cambien una decisión, filtro o revisión.
- Reutiliza una etiqueta estable en vez de crear variantes ortográficas.
- Comprueba los recuentos de uso antes de eliminar o consolidar etiquetas.
- Usa la privacidad cuando el propio nombre de la etiqueta revele un contexto sensible.
- Retira las etiquetas que ya no describan trabajo activo.

Si algo es el hogar principal y duradero del trabajo, conviértelo en una [categoría](./categories.mdx).
