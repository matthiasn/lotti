# Registra y valora tus sesiones de trabajo

Lotti registra el trabajo concentrado como entradas de diario cronometradas. Inicia un temporizador desde la entrada o tarea que contiene el trabajo para que su duración y contexto se mantengan juntos. Cuando termina la sesión, puedes añadir una valoración estructurada mientras la experiencia todavía está fresca.

El ejemplo de Project Waddle valora un ensayo de pase de lista de pingüinos emperador de 23 minutos. El panel de valoración que se muestra aquí es la interfaz de producción sobre ese registro de tiempo real, no una maqueta de documentación.

## Registra el tiempo en el contexto adecuado

Abre una entrada o tarea y usa su acción de temporizador. Un temporizador creado desde una tarea u otra entrada del diario se vincula a esa fuente, de modo que el trabajo circundante sigue visible en la actividad vinculada. La duración se actualiza mientras grabas y la acción de detener cierra la sesión en esa misma entrada de tiempo.

Usa un temporizador para un tramo coherente de trabajo. Si cambia el propósito —de inspeccionar el hábitat a la diplomacia de sardinas, por ejemplo— detén el temporizador actual e inicia otro bajo la tarea o categoría correcta. Así tanto el historial de entradas como el [Análisis de tiempo](./time-analysis.mdx) conservan sentido.

## Valora una sesión terminada

Después de terminar una sesión de al menos un minuto, Lotti muestra un indicador de estrella junto a la duración. Tócalo una vez: seguirá disponible hasta que guardes una valoración o reinicies la grabación. Para un temporizador mostrado dentro de actividad vinculada, el menú de acciones de la entrada también ofrece **Valorar sesión** o **Ver valoración**.

La valoración de sesión plantea cuatro preguntas:

- **Productividad** es una escala continua que va de improductivo a productividad máxima.
- **Energía** va del agotamiento a la energía plena.
- **Concentración** va de distracción constante a concentración ininterrumpida.
- **Este trabajo se sintió…** registra **Demasiado fácil**, **Justo bien** o **Demasiado desafiante**. **Justo bien** representa el equilibrio deseado entre dificultad y habilidad.

Arrastra o selecciona cualquier punto de una barra continua; los valores no están limitados a las marcas visibles. Añade una nota opcional para el contexto que los números no pueden expresar. **Guardar** permanece desactivado hasta que estén presentes las cuatro respuestas. **Omitir** cierra el panel sin guardar una valoración.

## Revisa y modifica una valoración

Una valoración guardada se almacena como su propia entrada de diario y se vincula a la entrada de tiempo que describe. Vuelve a abrir **Ver valoración** para cambiar las respuestas o la nota. El resumen guardado conserva la redacción de la pregunta y las opciones usadas entonces, por lo que las valoraciones sincronizadas siguen siendo comprensibles aunque el catálogo cambie más adelante.

Las valoraciones de sesión están disponibles cuando está activado el indicador de configuración **Valoraciones de sesión**. Un cliente que recibe un catálogo de valoraciones que no reconoce aún puede mostrar las respuestas almacenadas, pero mantiene esa valoración como solo lectura en vez de intentar adivinar cómo editar datos desconocidos.

## Usa las valoraciones como observaciones, no como notas

Las cuatro dimensiones describen una sesión, no el rendimiento personal. Una puntuación baja de energía puede explicar por qué un trabajo útil se sintió difícil; una concentración alta junto a **Demasiado fácil** puede indicar que la siguiente sesión necesita un alcance más ambicioso. Las notas son especialmente útiles para anomalías como una sardina voladora, una interrupción o una reunión de Control de misión inusualmente fluida.

Las valoraciones complementan la duración, no la cambian. El temporizador sigue siendo la fuente de cuánto duró el trabajo, mientras que la valoración registra cómo se sintió ese tiempo.
