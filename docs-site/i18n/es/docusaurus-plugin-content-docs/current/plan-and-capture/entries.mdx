# Añade entradas y contexto

Lotti usa un único menú **Añadir** para todo lo que capturas durante el día. Ábrelo desde el diario para crear una entrada independiente, o desde una tarea para que la nueva entrada quede vinculada a ella.

## Elige la entrada que corresponda al material

| Acción | Para qué usarla |
| --- | --- |
| **Lista de verificación** | Un conjunto breve de pasos concretos dentro de la tarea actual |
| **Tarea** | Un resultado independiente que necesita su propio estado y contexto |
| **Grabación de audio** | Una nota de voz que puedes volver a escuchar o transcribir |
| **Temporizador** | Una sesión de trabajo cronometrada vinculada al contexto actual |
| **Entrada de texto** | Notas, decisiones, observaciones o información pegada |
| **Importar imagen** | Una imagen existente del dispositivo |
| **Captura de pantalla** | Una nueva captura de escritorio adjunta como entrada de imagen |

El menú es contextual. **Lista de verificación** está disponible cuando una tarea puede ser dueña de los elementos, mientras que las acciones específicas del dispositivo, como **Captura de pantalla**, solo aparecen en las plataformas que las admiten.

## Conserva el contexto junto al trabajo

Al abrir **Añadir** desde una tarea, Lotti vincula la nueva entrada con ella. Así, grabaciones, notas, imágenes y registros de tiempo permanecen juntos en la página de detalle de la tarea. Usa una entrada independiente del diario cuando el material aún no pertenezca a un resultado concreto; podrás vincularla más tarde.

Consulta [Grabar audio](./recordings.mdx) para la captura y la transcripción, o [Trabajar con tareas](../organize-and-reflect/tasks.mdx) para conocer la estructura de tareas, las portadas y los enlaces entre tareas.
