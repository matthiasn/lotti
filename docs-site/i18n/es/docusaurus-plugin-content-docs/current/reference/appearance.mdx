# Elige la apariencia de Lotti

Abre **Ajustes → Temas** para controlar el aspecto de Lotti. El modo de
visualización y las dos elecciones de paleta son independientes, así que puedes
combinar un tema diurno luminoso con otro nocturno completamente distinto.

## Selecciona un modo de visualización

Elige uno de los tres modos:

- **Oscuro** siempre usa el tema oscuro seleccionado.
- **Automático** sigue el ajuste claro u oscuro actual del sistema operativo.
- **Claro** siempre usa el tema claro seleccionado.

El modo automático es útil cuando tu dispositivo cambia de apariencia según un
horario. Lotti sigue ese cambio y conserva la paleta elegida para cada lado.

## Elige las paletas clara y oscura

1. Selecciona **Tema claro** o **Tema oscuro**.
2. Elige una paleta con nombre en la hoja.
3. Repite el proceso para la otra apariencia si lo necesitas.

<ManualScreenshot

No hay botón Guardar. Las elecciones de modo de visualización y paleta se
guardan en cuanto las seleccionas. El interruptor claro y oscuro del propio
manual es independiente: cambia la documentación y elige automáticamente la
variante de captura correspondiente, pero no modifica tus ajustes de Lotti.
