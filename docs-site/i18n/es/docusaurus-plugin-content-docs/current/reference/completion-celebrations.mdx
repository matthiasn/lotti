# Personaliza las celebraciones de finalización

Abre **Ajustes → Ajustes avanzados → Animaciones** para decidir cómo responde
Lotti cuando terminas una tarea, completas un hábito o marcas un elemento de una
lista de comprobación.

## Elige dónde aparecen las celebraciones

- **Animaciones de celebración** es el interruptor maestro de los efectos
  visuales.
- **Hábitos**, **Elementos de listas de comprobación** y **Tareas** controlan de
  manera independiente sus efectos visuales de finalización.
- **Háptica de finalización** controla la breve respuesta física y sigue
  disponible aunque las animaciones estén desactivadas.

Desactivar una animación nunca revierte la finalización misma. Si el sistema
operativo solicita reducir el movimiento, Lotti suprime los efectos y conserva
los estilos elegidos para más tarde.

## Asigna un estilo a cada tipo de finalización

<ManualScreenshot
  caseId="celebrations/styles"
  alt="Selector de estilo de celebraciones de Lotti con Tareas seleccionado y las opciones Chispas, Fuegos artificiales, Confeti, Brasas, Burbujas, Aleatorio y Combinar dos"
  caption="Las tareas, los hábitos y los elementos de listas de comprobación pueden usar estilos distintos. Selecciona una tarjeta para obtener una vista previa y aplicarla."
/>

Para cada tipo de finalización, elige **Chispas**, **Fuegos artificiales**,
**Confeti**, **Brasas** o **Burbujas**. Elige **Aleatorio** para un estilo único
nuevo en cada finalización, o **Combinar dos** para superponer dos estilos
seleccionados al azar.

Usa **Pruébalo** para previsualizar la selección actual sin cambiar datos reales
de tareas ni hábitos.

## Ajusta un estilo individual

Selecciona la acción de personalizar de un estilo fijo para abrir su área de
pruebas a pantalla completa.

<ManualScreenshot
  caseId="celebrations/playground"
  alt="Área de pruebas de la celebración Chispas de Lotti con vista previa de tarea en directo y controles agrupados de Forma, Movimiento y Apariencia"
  caption="Los cambios se guardan y aplican en todas partes de inmediato. Repite la muestra después de cada ajuste."
/>

Los controles disponibles dependen del efecto y se agrupan en **Forma**,
**Movimiento** y **Apariencia**. Ajusta los controles deslizantes y usa después
la muestra destacada o **Repetir** para ver el resultado. **Restablecer valores
predeterminados** recupera el estilo integrado; la confirmación ofrece
**Deshacer** si el restablecimiento fue accidental.

La personalización es global para ese estilo. No se vincula a la tarea de ejemplo
mostrada en el área de pruebas ni a una tarea real concreta.
