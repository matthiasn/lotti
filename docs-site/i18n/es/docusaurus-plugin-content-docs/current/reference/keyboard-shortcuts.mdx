# Trabaja con atajos de teclado

Lotti de escritorio ofrece un catálogo con búsqueda de comandos globales y
contextuales. Abre **Ajustes → Atajos de teclado** para ver cada comando con la
combinación resuelta para tu plataforma.

## Abre rápidamente la ayuda de atajos

Pulsa **F1** o **Primaria + ?** para abrir la referencia de atajos sin abandonar
el trabajo actual. **Primaria** significa Comando en macOS y Control en Windows
o Linux.

Entre los comandos globales útiles con frecuencia están:

| Acción | Atajo |
| --- | --- |
| Abrir la paleta de comandos | Primaria + K |
| Crear una entrada de texto | Primaria + N |
| Crear una tarea | Primaria + T |
| Abrir Tareas, Daily OS, Proyectos o Hábitos | Primaria + 1 / 2 / 3 / 4 |
| Abrir Paneles, Diario, Eventos o Ajustes | Primaria + 5 / 6 / 7 / 8 |

Las páginas individuales añaden comandos contextuales, como guardar, actualizar,
buscar o crear. Su disponibilidad puede depender de qué tiene el foco o de si
hay cambios sin guardar.

## Ejecuta un comando sin abandonar tu trabajo

Pulsa **Primaria + K** para abrir la paleta de comandos sobre la página actual.
Combina acciones globales —crear entradas, abrir tareas, navegar y hacer zoom—
con los comandos que proporciona la página activa.

<ManualScreenshot
  caseId="keyboard-shortcuts/palette"
  alt="La paleta de comandos de Lotti abierta sobre Atajos de teclado, con los comandos disponibles de creación, navegación, vista y edición"
  caption="Solo aparecen los comandos disponibles en el contexto actual. La vista móvil muestra la misma paleta de escritorio en una ventana estrecha; los comandos de teclado son una función de escritorio."
/>

Empieza a escribir para limitar la lista por nombre de acción, grupo o
combinación de teclas. Usa las flechas arriba y abajo para mover el resaltado,
**Intro** para ejecutar la acción y **Escape** para cerrar la paleta. Puesto que
la paleta es contextual, un comando como **Guardar** solo aparece cuando el
editor con el foco puede guardar en ese momento.

## Busca por acción o tecla

<ManualScreenshot
  caseId="keyboard-shortcuts/search"
  alt="Catálogo de atajos de teclado de Lotti filtrado con una consulta de búsqueda"
  caption="Busca en el catálogo en vez de memorizarlo. Los resultados se actualizan mientras escribes."
/>

Usa el campo de búsqueda cuando conozcas la acción o parte de su combinación de
teclas. El catálogo de la aplicación es la fuente de autoridad si una
combinación cambia en una versión posterior de Lotti.
