# Elige un estilo de grabación

Abre **Ajustes → Estilo de grabación** para elegir la visualización que se
muestra mientras Lotti graba audio. Esto cambia solo la presentación; ambos
estilos representan el mismo proceso de grabación.

## Compara los dos estilos

- **Moderno** muestra un orbe de energía animado que reacciona al nivel actual.
- **Analógico** muestra un medidor VU tradicional con una aguja móvil.

Selecciona una de las tarjetas para convertirla en el estilo activo. No hay una
acción independiente de Guardar o Continuar.

## Previsualiza con tu voz

Activa **Probar con tu voz** para dirigir la vista previa desde el micrófono.
Habla a la distancia a la que esperas usarlo durante una grabación y observa cómo
responde la visualización elegida. Desactiva la opción cuando termines de
comparar estilos.

La vista previa te ayuda a elegir un lenguaje visual; no es una herramienta de
calibración de ganancia de entrada ni de calidad de grabación. Consulta
[Grabar audio](../plan-and-capture/recordings.mdx) para ver el flujo de trabajo
de grabación real.
