# Encuentra los ajustes

Los ajustes se agrupan por la capacidad que controlan. El escritorio usa un árbol
persistente con la página seleccionada al lado; el móvil usa una lista centrada y
abre una página de ajustes cada vez.

Elige **Manual** en la sección final diferenciada de Ajustes para abrir esta guía
en tu navegador predeterminado. En macOS, **Ayuda → Manual** ofrece el mismo
enlace. Por defecto, la guía sigue el idioma del sistema; elige otro idioma en
**Ajustes → Ajustes avanzados → Idioma** cuando lo necesites.

Los destinos habituales incluyen:

- **[Apariencia](./appearance.mdx)** para los modos de visualización claro,
  oscuro y automático, además de paletas de tema independientes.
- **[Estilo de grabación](./recording-style.mdx)** para el orbe de energía o el
  medidor analógico que se muestra durante una grabación.
- **[Voz](./speech.mdx)** para la voz en el dispositivo, el modelo y la velocidad
  usados para leer resúmenes de tareas en voz alta.
- **[Atajos de teclado](./keyboard-shortcuts.mdx)** para el catálogo de comandos
  de escritorio con búsqueda.
- **[Celebraciones de finalización](./completion-celebrations.mdx)** para
  animaciones, háptica, estilos y su área de pruebas en directo.
- **[Ajustes avanzados](./advanced-settings.mdx)** para indicadores de
  configuración, registro, acciones de mantenimiento, diagnósticos de
  incorporación e información de compilación.
- **Categorías** para límites de planificación y **Etiquetas** para etiquetas
  transversales.
- **Hábitos** y **Paneles** para estructuras de reflexión.
- **Ajustes de IA** para proveedores, modelos, perfiles y habilidades.
- **Daily OS** para el perfil de inferencia y el saludo del planificador.
- **Sync** para el estado de conexión y actividad.

El conjunto exacto puede variar según las capacidades de la plataforma y las
funciones activadas.

## Gestiona las definiciones

Abre **Ajustes → Definiciones** para gestionar las estructuras que organizan tus
datos. Las categorías fijan límites de planificación y privacidad, las etiquetas
marcan tareas entre categorías, los hábitos definen rutinas recurrentes, los
paneles organizan vistas de reflexión y los medibles definen los números de los
que haces seguimiento.
