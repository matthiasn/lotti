# Elige la voz de lectura

Cuando la lectura en voz alta esté disponible, abre **Ajustes → Voz**. Estos
controles se aplican a los resúmenes de IA de las tareas; no cambian la captura
del micrófono ni qué proveedor de IA transcribe una grabación.

## Elige una voz

Usa el selector **Femenina / Masculina** para explorar los dos grupos y, después,
selecciona una de las cinco voces numeradas de ese grupo. Cambiar de grupo solo
modifica la lista visible; selecciona una voz numerada para activarla.

La voz seleccionada es una preferencia local del dispositivo. Así, cada
dispositivo puede usar una voz que se adapte a sus altavoces y entorno de escucha
sin cambiar la elección de tus otros dispositivos.

## Descarga del modelo y privacidad

Lotti usa actualmente **Supertonic 3** para leer en voz alta. La síntesis se
ejecuta en el dispositivo en vez de enviar el resumen a un servicio de voz en la
nube. El modelo es grande y se descarga la primera vez que se necesita, así que
prevé tiempo, ancho de banda de red y almacenamiento para esa configuración
inicial.

## Establece la velocidad de lectura

Elige una velocidad entre **0,5×** y **2×**. El ajuste **1×** es la velocidad
natural; los valores inferiores ralentizan el informe y los superiores lo
acortan. La velocidad elegida también se guarda solo en el dispositivo actual.

El botón de lectura en voz alta aparece solo cuando la función está activada, la
plataforma admite el motor de voz y la tarea tiene un resumen de IA no vacío.

Para la entrada hablada y los resultados de reconocimiento guardados, consulta
[Grabar audio](../plan-and-capture/recordings.mdx).
