# Importa datos de salud

La importación de salud está disponible en iOS y Android. Abre
**Ajustes → Avanzado → Importación de salud**, elige una fecha de inicio y una de
fin y, después, selecciona la familia de datos que quieres que Lotti lea del
almacén de datos de salud del teléfono.

Las importaciones disponibles son:

- **Actividad** para totales de actividad;
- **Sueño** para muestras de sueño;
- **Frecuencia cardíaca** para observaciones de pulso;
- **Presión arterial** para lecturas de presión arterial;
- **Medición corporal** para lecturas de métricas corporales; y
- **Entrenamiento** para entrenamientos registrados.

Cada botón importa solo su propia familia de datos dentro del intervalo de fechas
mostrado. Cambiar el intervalo no inicia una importación por sí solo.

Tu sistema operativo puede pedir permiso para los datos de salud. Lotti solo
puede leer los tipos que autorices. Si falta una familia de datos esperada,
comprueba los permisos de salud de la aplicación en los ajustes del sistema del
teléfono y vuelve a intentar esa importación.

La importación de salud no se muestra en el escritorio porque los almacenes de
salud móviles subyacentes no están disponibles allí. Las observaciones importadas
pueden seguir viéndose en los paneles relevantes de Lotti después de
sincronizarse con otro dispositivo.
