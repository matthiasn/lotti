## I blocchi dell'edificio, senza fingere di essere tutti uguali

| Blocco di costruzione | Usalo per | Esempio |
| --- | --- | --- |
| Categoria | Un'area significativa di vita o di lavoro | Salute, Famiglia, Ricerca |
| Compito | Un risultato che ha bisogno di progresso | Preparare il rilascio V1 |
| Lista di controllo | Passi concreti all'interno di un'attività | Verificare le note di rilascio |
| Entrata | Contesto o prova | Nota, immagine, registrazione audio |
| Blocco pianificato | Una fessura proposta nel giorno | Recensione di rilascio, 10:00-11:00 |
| Tempo record | Che cosa è successo? | Recensione di rilascio, 10:18–11:07 |
| Abitazioni | Un comportamento ricorrente si valuta | Prendere farmaci |
| Misurazione | Un'osservazione numerica | Frequenza cardiaca a riposo |
| Dashboard | Una vista sulle misure utili | Panoramica della formazione |

## Gli agenti sono ospiti, non co-proprietari

Un agente può leggere il contesto che si dà, formare un parere, riassumere una disordinata
e proporre una modifica successiva utile. La sua relazione è un'opinione
prova, non un nuovo fatto nella vostra storia. Compito proposto, lista di controllo, stato,
o le modifiche della data vi aspettano per confermarle o licenziarle.
sono un titolo iniziale o una lingua per un compito altrimenti vuoto, dopo di che,
si applica la stessa regola di approvazione umana.

## Una prima orbita pratica

1. Crea categorie che usi veramente per prendere decisioni.
2. Acquisire il lavoro come compiti e mantenere il materiale di supporto attaccato a loro.
3. Usa il sistema operativo giornaliero per decidere cosa appartiene in un giorno particolare.
4. Registra il tempo reale e le osservazioni senza costringerle a corrispondere al piano.
5. Valutare le abitudini e i cruscotti per i modelli piuttosto che i singoli perfetti
   giorni.

## Mantenere la proprietà chiara

- Un'attività collegata possiede il suo titolo e la sua categoria.
- Un blocco giornaliero del sistema operativo possiede il suo posizionamento e la durata.
- Un'entrata del tempo possiede il record di quello che è successo.
- AI propone; si esamina e si decide.

:::tip[Il punto non è una linea temporale immacolata]

Se il piano è cambiato, lascia che cambi. Se il giorno è andato lateralmente, registrare ciò che ha fatto
un sistema utile ti aiuta a vedere la forma della tua vita; non chiede
si per eseguire uno più produttivo per il database.

:::
