# Lavora con le scorciatoie della tastiera

Desktop Lotti espone un catalogo ricercabile di app-wide e contestuale
comandi. Aprire **Impostazioni → Scorciatoie tastiera** per vedere ogni comando con il
vincolante risolto per la vostra piattaforma.

## Aprire la scorciatoia aiuto rapidamente

Premere **F1** o **Primary + ?** per aprire il riferimento di scorciatoia senza lasciare
**Primary** significa Comando su macOS e Controllo su Windows o
Linux.

I comandi app-wide frequentemente utili includono:

| Azione | Scorciatoia |
| --- | --- |
| Aprire la tavolozza dei comandi | Primario + K |
| Creare una voce di testo | Primario + N |
| Creare un'attività | Primario + T |
| Attività aperte, sistema operativo giornaliero, progetti o abitudini | Primario + 1 / 2 / 3 / 4 |
| Aprire Dashboard, Journal, Eventi o Impostazioni | Primario + 5 / 6 / 7 / 8 |

Le singole pagine aggiungono comandi contestuali come salvare, aggiornare, cercare o
creare. La loro disponibilità può dipendere da ciò che è attualmente focalizzato o se
ci sono cambiamenti non salvati.

## Eseguire un comando senza lasciare il tuo lavoro

Premere **Primary + K** per aprire la palette di comando sulla pagina corrente.
combina le azioni app-wide—creare le voci, aprire i compiti, navigare e
zoom—con comandi forniti dalla pagina attiva.

<ManualScreenshot
  caseId="keyboard-shortcuts/palette"

Inizia a digitare per restringere l'elenco per nome d'azione, gruppo o combinazione di tasti.
le frecce su e giù per spostare l'evidenziatore, **Enter** per eseguirlo, e
**Escape** per chiudere la tavolozza, perché la tavolozza è contestuale, un comando
come **Save** appare solo quando l'editor focalizzato può attualmente salvare.

Utilizzare il campo di ricerca quando si conosce l'azione o parte della sua chiave
combinazione. Il catalogo in-app è l'autorità se un cambiamento vincolante in un
versione successiva di Lotti.
