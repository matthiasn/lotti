# Scegli uno stile di registrazione

Apri **Impostazioni → Recording Style** per scegliere la visualizzazione mostrata mentre
Lotti registra l'audio. Questo cambia solo la presentazione; entrambi gli stili rappresentano il
stesso processo di registrazione.

## Confronta i due stili

- **Modern** mostra un orb di energia animata che reagisce al livello attuale.
- **Analogue** mostra un metro VU tradizionale con un ago in movimento.

Selezionare una scheda per renderlo lo stile attivo. Non c'è nessun Salva separato o
Continuare l'azione.

## Anteprima con la tua voce

Abilitare **Ti con la tua voce** per guidare l'anteprima dal microfono.
alla distanza che ti aspetti di utilizzare durante la registrazione e guardare come il selezionato
Risponde visuale. Spegni l'opzione quando hai finito di confrontare gli stili.

L'anteprima ti aiuta a scegliere un linguaggio visivo; non è un input-gain o
strumento di calibrazione di qualità di registrazione. Vedi [Record audio](../plan-and-capture/recordings.mdx)
per il flusso di lavoro di registrazione reale.
