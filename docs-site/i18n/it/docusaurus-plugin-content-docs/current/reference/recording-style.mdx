# Scegli uno stile di registrazione

Apri **Impostazioni → Recording Style** per scegliere la visualizzazione mostrata mentre
Lotti registra l'audio. Questo cambia solo la presentazione; entrambi gli stili rappresentano il
stesso processo di registrazione.

## Confronta i due stili

- **Modern** mostra un orb di energia animata che reagisce al livello attuale.
- **Analogue** mostra un metro VU tradizionale con un ago in movimento.

Seleziona una scheda per renderla lo stile attivo. Non c'è nessuna azione Salva o
Continua separata.

## Anteprima con la tua voce

Abilita **Prova con la tua voce** per guidare l'anteprima dal microfono alla distanza che
prevedi di usare durante la registrazione e osserva come risponde lo stile visivo selezionato.
Disattiva l'opzione quando hai finito di confrontare gli stili.

L'anteprima ti aiuta a scegliere un linguaggio visivo; non è uno strumento di calibrazione
del guadagno in ingresso o della qualità di registrazione. Vedi [Registrare l'audio](../plan-and-capture/recordings.mdx)
per il flusso di lavoro di registrazione reale.
