# Trova le impostazioni

Le impostazioni sono raggruppate dalla capacità di controllo. Desktop utilizza un persistente
albero con la pagina selezionata accanto a esso; mobile utilizza un elenco focalizzato e apre uno
Impostazioni pagina alla volta.

Scegliere **Manual** nella sezione finale distinta di Impostazioni per aprire questa guida
Su macOS, **Help → Manual** fornisce lo stesso link.
La guida segue la lingua del sistema per impostazione predefinita; scegli una lingua diversa
in ** Impostazioni → Impostazioni avanzate → Lingua** quando necessario.

Le destinazioni comuni includono:

- **[Apparenza](./appearance.mdx)** per display leggero, scuro e automatico
  modalità più tavolozze a tema indipendenti.
- **[Stile di registrazione](./recording-style.mdx)** per l'energia-orbo o analogico
  metro mostrato durante la registrazione.
- **[Speech](./speech.mdx)** per la voce on-device, il modello e la velocità utilizzata per
  leggere i riassunti delle attività ad alta voce.
- **[Keyboard scorciatoie](./keyboard-shortcuts.mdx)** per il desktop ricercabile
  catalogo dei comandi.
- **[Paesaggi di completamento](./completion-celebrations.mdx)** per animazioni,
  haptics, stili e il loro parco giochi dal vivo.
- **[Impostazioni avanzate](./advanced-settings.mdx)** per bandiere, registrazione,
  operazioni di manutenzione, diagnostica di bordo, e costruire informazioni.
- **Categories** per la pianificazione dei confini e **Labels** per i tag cross-cutting.
- Per le strutture di riflessione.
- ** Impostazioni AI** per fornitori, modelli, profili e competenze.
- **Daily OS** per il profilo di inferenza del pianificatore e il saluto.
- **Sync** per la connessione e lo stato di attività.

Il set esatto può variare con le funzionalità della piattaforma e le funzionalità abilitate.

## Gestione delle definizioni

Aprire **Impostazioni → Definizioni** per gestire le strutture che organizzano la tua
categorie impostare la pianificazione e i confini della privacy, etichette tag attività attraverso
categorie, abitudini definire routine ricorrenti, cruscotti organizzare riflessione
viste, e misurabili definiscono i numeri che si tracciano nel tempo.
