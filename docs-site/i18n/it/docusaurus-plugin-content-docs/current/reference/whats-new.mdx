# Sfoglia ciò che è nuovo

What's New presenta note di rilascio all'interno di Lotti, abbinate alla versione che sei
in esecuzione. Può aprirsi automaticamente dopo un primo lancio o versione ammissibili
cambiamento, ed è disponibile anche dal **What's New** voce in Impostazioni quando
che la capacità è abilitata.

Le note di rilascio sono ordinate prima di tutto nuove. Ogni pagina combina una specifica di rilascio
immagine eroe, la sua versione app esatta, e un breve conto dell'utente-visibile
cambiamenti.

## Leggi il comunicato corrente

Il badge della versione identifica il rilascio le cui note sono sullo schermo.
segna il nuovo rilascio nel set corrente; non significa ogni elemento sul
pagina è incompiuta o sperimentale.

Il corpo scorre in modo indipendente quando il rilascio è più lungo di quello disponibile
altezza. I collegamenti utilizzano il normale gestore del collegamento Lotti, e le immagini sono caricate in anticipo
la pagina cambia quando disponibile in modo da muoversi tra le versioni rimane liscia.

## Spostare la storia del rilascio

Utilizzare i comandi del piè di pagina per navigare:

- il giusto chevron apre la prossima versione più vecchia;
- il chevron sinistro ritorna ad un nuovo rilascio;
- i marcatori di posizione mostrano dove la pagina corrente si trova nel set caricato;
- **Skip** chiude un set multi-release prima dell'ultima pagina e segna l'intero
  set caricato come visto; e
- **Done** appare sulla pagina più antica e segna anche l'intero set come visto.

Chiusura del modale trascinandolo via o licenziando la barriera circostante è
più conservatore: Lotti segna solo le uscite che hai effettivamente raggiunto, attraverso
la pagina più lontana visualizzata. Le uscite che non hai raggiunto possono quindi apparire
La prossima volta che si apre il nuovo.

## Rivisitare le note già respinte

Quando non rimangono note non lette, l'apertura di What's New mostra lo stato **Sei in pari**.
Scegli **Visualizza le versioni precedenti** per cancellare i marcatori locali visti e
sfogliare nuovamente il set di rilascio disponibile. Questo cambia solo la nota locale di rilascio
storia; non altera la versione dell'app installata o scarica un'altra build.

What's New non mostra mai note per un rilascio più recente rispetto al Lotti installato
versione. Se il contenuto di rilascio non può essere caricato, l'applicazione rientra in un vuoto
lo stato piuttosto che presentare le note per la costruzione sbagliata.

Per il numero esatto di costruzione, la piattaforma, i totali di entrata locale e i conteggi delle attività, vedi
[Informazioni su Lotti](./advanced-settings.mdx#verificare-la-costruzione-e-i-totali-locali).
