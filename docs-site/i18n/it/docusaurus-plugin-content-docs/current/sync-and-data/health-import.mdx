# Importare i dati sulla salute

Health Import è disponibile su iOS e Android. Apri **Impostazioni → Avanzate →
Health Import**, scegliere una data di inizio e fine, quindi selezionare la famiglia di dati che si
Voglio che Lotti legga dal negozio di dati sanitari del telefono.

Le importazioni disponibili sono:

- **Attività** per i totali di attività;
- **Sleep** per i campioni di sonno;
- ** Tasso cardiaco** per le osservazioni del polso;
- ** Pressione di sangue** per letture di pressione sanguigna;
- **Misura corporea** per misurazioni quali metriche del corpo; e
- **Lavorare** per gli allenamenti registrati.

Ogni pulsante importa solo la propria famiglia di dati sull'intervallo di date visualizzato.
Cambiare la gamma non inizia un'importazione da sola.
