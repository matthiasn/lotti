# Voeg vermeldingen en context toe

Lotti gebruikt één **Toevoegen**-menu voor de dingen die je gedurende de dag vastlegt. Open het
vanuit het journaal voor een zelfstandig item, of vanuit een taak om het nieuwe item te behouden
aan die taak verbonden.

## Kies de inzending die bij het materiaal past

| Actie | Gebruik het voor |
| --- | --- |
| **Checklist** | Een korte reeks concrete stappen binnen de huidige taak |
| **Taak** | Een aparte uitkomst die een eigen status en context nodig heeft |
| **Audio-opname** | Een spraaknotitie die kan worden afgespeeld of getranscribeerd |
| **Timer** | Een getimede werksessie gekoppeld aan de huidige context |
| **Tekstinvoer** | Aantekeningen, beslissingen, observaties of geplakte informatie |
| **Afbeelding importeren** | Een bestaande afbeelding van het apparaat |
| **Schermafbeelding** | Een nieuwe desktopscreenshot bijgevoegd als afbeeldingsinvoer |

Het menu is contextueel. **Checklist** is beschikbaar wanneer een taak eigenaar kan zijn van de
items, terwijl apparaatspecifieke acties zoals **Screenshot** alleen verschijnen waar
het platform ondersteunt hen.

## Houd de context bij het werk

Wanneer u **Toevoegen** opent vanuit een taak, koppelt Lotti het nieuwe item terug aan die taak.
Dat houdt opnames, notities, afbeeldingen en tijdregistraties bij elkaar voor de taak
detailpagina. Gebruik een zelfstandige journaalboeking als het materiaal er niet bij hoort
tot een specifiek resultaat nog; het kan later worden gekoppeld.

Zie [Audio opnemen](./recordings.mdx) voor vastleggen en transcriptie, of
[Werken met taken](../organize-and-reflect/tasks.mdx) voor taakstructuur, omslag
kunst- en taaklinks.
