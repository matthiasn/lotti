# Kies een opnamestijl

Open **Instellingen → Opnamestijl** om de getoonde visualisatie te kiezen
Lotti neemt audio op. Dit verandert alleen de presentatie; beide stijlen vertegenwoordigen de
hetzelfde opnameproces.

## Vergelijk de twee stijlen

- **Modern** toont een geanimeerde energiebol die reageert op het huidige niveau.
- **Analoog** toont een traditionele VU-meter met een bewegende naald.

Selecteer een van beide kaarten om er de actieve stijl van te maken. Er is geen afzonderlijk Opslaan of
Ga door met actie.

## Bekijk een voorbeeld met je stem

Schakel **Probeer met je stem** in om het voorbeeld via de microfoon aan te sturen. Spreek
op de afstand die u verwacht te gebruiken tijdens het opnemen en kijk hoe u selecteert
visueel reageert. Schakel de optie uit als u klaar bent met het vergelijken van stijlen.

De preview helpt je bij het kiezen van een beeldtaal; het is geen input-winst of
Kalibratietool voor opnamekwaliteit. Zie [Audio opnemen](../plan-and-capture/recordings.mdx)
voor de daadwerkelijke opnameworkflow.
