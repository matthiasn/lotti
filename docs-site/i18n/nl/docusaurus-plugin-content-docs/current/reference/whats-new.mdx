# Blader door wat er nieuw is

What's New presenteert release-opmerkingen in Lotti, afgestemd op de versie waarin u zich bevindt
rennen. Het kan automatisch worden geopend na een in aanmerking komende eerste lancering of versie
wijzigen, en het is ook beschikbaar via het item **Wat is er nieuw** in Instellingen wanneer
die mogelijkheid is ingeschakeld.

Releaseopmerkingen worden als nieuwste eerst gerangschikt. Elke pagina combineert een release-specifiek
hero-afbeelding, de exacte app-versie en een kort verslag van de gebruiker zichtbaar
veranderingen.

## Lees de huidige release

De versiebadge identificeert de release waarvan de opmerkingen op het scherm verschijnen. **NIEUW**
markeert de nieuwste release in de huidige set; het betekent niet dat elk item op de
pagina is nog niet voltooid of experimenteel.

Het lichaam scrollt onafhankelijk wanneer de release langer is dan beschikbaar
hoogte. Links gebruiken de normale Lotti-linkhandler en afbeeldingen worden vooraf geladen
beschikbaar, zodat het schakelen tussen releases soepel blijft.

Gebruik de voettekstknoppen om te navigeren:

- de rechter chevron opent de volgende oudere release;
- de linker chevron keert terug naar een nieuwere versie;
- de positiemarkeringen geven aan waar de huidige pagina zich in de geladen set bevindt;
- **Skip** sluit een multi-releaseset vóór de laatste pagina en markeert het geheel
  geladen set zoals gezien; en
- **Klaar** verschijnt op de oudste pagina en markeert ook de hele set als gezien.

Het modale venster sluiten door het weg te slepen of door het gedimde gebied eromheen te selecteren
conservatiever: Lotti markeert alleen de releases die je daadwerkelijk hebt bereikt
de verste bekeken pagina. Releases die u niet heeft bereikt, kunnen daarom verschijnen
de volgende keer dat What's New wordt geopend.

## Bekijk notities die je al hebt afgewezen opnieuw

Als er geen ongeziene releases meer zijn, toont de opening van What's New een **Jullie zijn allemaal betrapt
omhoog** staat. Kies **Bekijk eerdere releases** om de lokaal geziene markeringen te wissen en
blader opnieuw door de beschikbare releaseset. Dit verandert alleen de lokale release-opmerking
geschiedenis; het verandert de geïnstalleerde app-versie niet en downloadt geen andere build.

What's New toont nooit notities voor een release die nieuwer is dan de geïnstalleerde Lotti
versie. Als de release-inhoud niet kan worden geladen, valt de app terug op een lege status
staat in plaats van aantekeningen te maken voor de verkeerde build.

Zie voor het exacte buildnummer, platform, lokale invoertotalen en taakaantallen
[Over Lotti](./advanced-settings.mdx#controleer-de-build-en-lokale-totalen).
