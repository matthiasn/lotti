# Gezondheidsgegevens importeren

Gezondheid Import is beschikbaar op iOS en Android. Open **Instellingen → Geavanceerd →
Gezondheid Importeren**, kies een begin- en einddatum en selecteer vervolgens de gegevensfamilie die u gebruikt
wil dat Lotti leest uit de gezondheidsgegevensopslag van de telefoon.

De beschikbare importproducten zijn:

- **Activiteit** voor activiteitentotalen;
- **Slaap** voor slaapmonsters;
- **Hartslag** voor polswaarnemingen;
- **Bloeddruk** voor bloeddrukmetingen;
- **Lichaamsmeting** voor lichaamsmetrische metingen; en
- **Training** voor opgenomen trainingen.

Elke knop importeert alleen zijn eigen gegevensfamilie over het weergegeven datumbereik.
Het wijzigen van het bereik start niet vanzelf een import.

Uw besturingssysteem kan om toestemming voor gezondheidsgegevens vragen. Lotti kan alleen lezen
de soorten die u toekent. Als een verwachte gegevensfamilie ontbreekt, controleer dan die van de app
gezondheidsmachtigingen in de systeeminstellingen van de telefoon en probeer het importeren opnieuw.

Health Import wordt niet weergegeven op desktop vanwege de onderliggende mobiele gezondheid
winkels zijn daar niet beschikbaar. Geïmporteerde waarnemingen kunnen nog steeds worden bekeken
de relevante Lotti-dashboards nadat ze met een ander apparaat zijn gesynchroniseerd.
