# Crie sua primeira tarefa

Uma tarefa é o local para um resultado, sua lista de verificação e as notas, gravações,
imagens ou registros de tempo que explicam o trabalho.

## Comece pelo caminho mais rápido

Se você ainda estiver na integração, fale o resultado ou escolha **Prefere digitar?**.
Escolha a área à qual a tarefa pertence; Lotti usa essa categoria para manter a
tarefa visível nas visualizações certas e para selecionar a automação configurada.

de sair da integração. Abra o cartão para continuar no espaço de trabalho
normal de tarefas.

## Crie uma tarefa a partir da lista de tarefas

1. Abra **Tarefas**.
2. Use a ação **+** e escolha **Tarefa**.
3. Escreva um título orientado para os resultados.
4. Escolha uma categoria quando houver uma claramente aplicável.
5. Salve a tarefa.

No desktop, o espaço de trabalho de tarefas mantém a lista e a tarefa
selecionada lado a lado. No celular, selecionar uma tarefa abre a visualização
de detalhes em tela inteira. Em ambos os layouts, a arte de capa das tarefas e
o contexto da categoria facilitam percorrer uma lista cheia.

## Adicione contexto útil

Abra a tarefa e use a ação de adicionar para anexar uma nota, gravação, imagem
ou outra entrada. Mantenha o material junto da tarefa quando isso puder ajudar
você – ou um assistente de IA – a entender a próxima ação mais tarde.

## Torne-o acionável

Adicione uma pequena lista de verificação com etapas concretas. Se um provedor
de IA e as habilidades relevantes estiverem configurados, Lotti também pode
transformar uma gravação ou um contexto escrito em sugestões de lista de
verificação. Revise as alterações propostas antes de aplicá-las.

## Decida quando é importante

- Defina uma data de vencimento quando houver um prazo real.
- Defina prioridade para expressar urgência.
- Traga a tarefa para um check-in do Daily OS quando ela fizer parte do plano
  de hoje.

Continue com [gerenciamento de tarefas](../organize-and-reflect/tasks.mdx) para prioridade,
filtragem e fluxos de trabalho assistidos por voz.
