# Lembre-se de eventos significativos

Os eventos dão aos momentos significativos um lugar próprio na Lotti. Use um
evento para um lançamento, uma viagem, uma comemoração, uma conferência ou
outro momento que você queira revisitar. Use tarefas para o trabalho de
preparação e acompanhamento em torno dele.

Os eventos são atualmente um recurso opcional. Abra **Configurações → Avançado → Sinalizadores**
e ative **Ativar eventos**. O destino **Eventos** aparece então na
barra lateral do computador e em **Mais** no celular.

## Navegue e filtre seus eventos

A visão geral é guiada por fotos, para que um evento seja reconhecível antes
mesmo de você ler o título. O evento futuro mais próximo aparece no destaque
**Próximos**. Os eventos passados são agrupados por ano, do mais recente para o
mais antigo.

Use **Pesquisar eventos** para encontrar um evento pelo texto. Selecione um
chip de categoria — como **Operações Penguin**, **Controle de Missão** ou
**Diplomacia de Peixes** — para restringir o arquivo completo. A filtragem se
aplica à coleção de eventos armazenada, não apenas aos cartões visíveis na tela
no momento.

A Lotti carrega um arquivo longo em páginas e renderiza apenas as linhas
próximas, então centenas de eventos cheios de fotos continuam práticos sem
carregar todas as capas de uma vez.

## Crie um evento

Selecione **Novo evento**. Um novo evento abre na sua página de detalhes, onde
você pode adicionar título, categoria, data e hora, status, foto de capa e
entradas de apoio.

Escolha um título que nomeie o momento, e não a sua preparação. Por exemplo,
**Gala de lançamento do Project Waddle** é o evento; **Recalibrar o alimentador
de peixes com gravidade zero** é uma tarefa vinculada.

O destaque mantém juntos a identidade do evento e os controles principais:

- selecione o título para editá-lo;
- selecione a pílula de categoria ou de status para alterá-la;
- selecione a data e hora para remarcar o evento;
- selecione uma estrela para alterar a classificação de um evento concluído; e
- abra o menu flutuante para alterar a capa ou excluir o evento.

As alterações são salvas pela própria página de detalhes; não há um formulário
separado para reabrir. A classificação aparece depois que um evento é
concluído, ou sempre que o evento já tiver uma classificação.

O resumo usa a resposta de IA vinculada mais recente, quando existe uma. Caso
contrário, mostra a própria nota do evento. Trate-o como uma recapitulação
rápida e depois use as fotos e a linha do tempo para o registro subjacente.

## Construa um registro fotográfico

A seção **Fotos** reúne todas as imagens vinculadas ao evento em ordem
cronológica. Selecione uma miniatura para abrir o visualizador em tela cheia,
depois deslize entre as fotos ou faça o gesto de pinça para ampliar. Uma
coleção grande é limitada a uma prévia compacta; o último bloco mostra quantas
outras fotos estão disponíveis.

A capa é uma das fotos vinculadas ao evento. Se nenhuma capa tiver sido
escolhida, a Lotti pode usar a primeira foto vinculada. Use **Alterar capa** no
menu flutuante para escolher outra imagem ou adicionar uma nova foto pelo mesmo
seletor.

## Siga o cronograma e trabalhos relacionados

  caption="Fotos, notas, gravações e tarefas de acompanhamento permanecem conectadas ao evento sem transformar o evento em si em uma tarefa."

A linha do tempo ordena as entradas vinculadas da mais antiga para a mais
recente. Ela pode conter:

- fotos e suas legendas;
- notas;
- entradas de áudio e notas de voz; e
- registros de tempo, mostrados como início, fim e tempo decorrido.

Selecione um item da linha do tempo para abrir sua entrada de origem. Selecione
**Adicionar** ao lado de **Linha do tempo** para criar uma nota, foto, entrada
de áudio ou outra entrada suportada, que fica automaticamente vinculada a este
evento.

A seção **Tarefas** é para preparação e acompanhamento. Selecione uma tarefa
existente para abri-la, ou selecione **Adicionar** para criar uma tarefa já
vinculada ao evento. No desktop, as tarefas permanecem visíveis em uma coluna à
direita enquanto você lê o evento; no celular, elas vêm depois da linha do
tempo, na mesma rolagem.

## Evento ou tarefa?

Use esta distinção ao decidir onde algo pertence:

| Use um evento para… | Use uma tarefa para… |
| --- | --- |
| Um momento com data, fotos, recapitulação e valor de memória | Uma ação com status, estimativa, lista de verificação ou sessão de trabalho |
| Gala de lançamento do Project Waddle | Recalibrar o alimentador de peixes com gravidade zero |
| Cúpula de futuros de sardinha de Europa | Confirmar as cápsulas interplanetárias de carga de sardinha |

Vincule os dois quando o trabalho apoiar o momento. Assim, o evento continua
agradável de revisitar, enquanto seu trabalho operacional permanece acionável.
