# Marque o trabalho com rótulos

Os rótulos adicionam uma segunda camada flexível de organização. Uma tarefa tem
uma única categoria principal, mas pode carregar vários rótulos para
iniciativas, urgência, estados de espera ou outra propriedade que atravesse os
limites das categorias.

O espaço de trabalho Intergalactic Penguin Logistics usa rótulos deliberadamente:

- **Project Waddle** conecta o trabalho de lançamento entre Operações Penguin,
  Controle de Missão e Diplomacia dos Peixes.
- **Habitat crítico** identifica o trabalho que precisa terminar antes de a
  expedição entrar no habitat orbital.
- **Aguardando Controle da Missão** registra um estado de bloqueio real.
- **Sardinha sensível ao mercado** mantém identificável o trabalho privado de
  negociação.

## Crie e edite um rótulo

1. Abra **Configurações → Etiquetas**.
2. Selecione **Criar etiqueta**.
3. Adicione um nome conciso e uma descrição opcional.
4. Escolha uma cor que continue distinguível ao lado dos outros rótulos.
5. Decida se o rótulo é privado e quais categorias podem usá-lo.
6. Salve.

Vale a pena escrever descrições quando um rótulo pode ser interpretado de mais
de uma maneira. Elas fornecem o significado operacional sem forçar essa
explicação em cada título de tarefa.

**Categorias aplicáveis** restringem onde o rótulo pode ser selecionado. O
Project Waddle está disponível nas três partes da expedição envolvidas no
lançamento, enquanto um rótulo especializado pode ficar limitado a uma
categoria. Isso mantém os seletores relevantes, em vez de apresentar todos os
rótulos em todos os lugares.

## Aplicar rótulos às tarefas

Abra uma tarefa e use o controle de rótulos dela para adicionar ou remover
rótulos. Os rótulos funcionam junto com os filtros de status, prioridade e
categoria da tarefa, então você pode fazer perguntas focadas como “mostrar o
trabalho aberto P1 crítico para o habitat” sem inventar uma nova categoria para
essa visão temporária.

Remover um rótulo de uma tarefa é tratado como uma rejeição intencional quando
as sugestões automáticas de rótulos estão ativadas. A Lotti registra essa
escolha para que a mesma sugestão não seja reaplicada imediatamente sem um novo
motivo.

## Mantenha os rótulos úteis

- Prefira rótulos que mudem uma decisão, um filtro ou uma revisão.
- Reutilize um rótulo estável em vez de criar variantes de grafia.
- Verifique as contagens de uso antes de excluir ou consolidar rótulos.
- Use a privacidade quando o próprio nome do rótulo revelar um contexto
  confidencial.
- Aposente rótulos que já não descrevem trabalho em andamento.

Se algo for o lar principal e duradouro do trabalho, transforme-o em uma
[categoria](./categories.mdx).
