# Personalize as celebrações de conclusão

Abra **Configurações → Configurações avançadas → Animações** para decidir como
o Lotti responde quando você termina uma tarefa, completa um hábito ou marca um
item de lista de verificação.

## Escolha onde as celebrações aparecem

- **Animações de celebração** é o interruptor mestre dos floreios visuais.
- **Hábitos**, **Itens da lista de verificação** e **Tarefas** controlam de
  forma independente seus efeitos visuais de conclusão.
- **Sensação tátil de conclusão** controla a resposta física curta e permanece
  disponível mesmo quando as animações estão desativadas.

Desativar uma animação nunca reverte a conclusão em si. Se o sistema
operacional solicitar movimento reduzido, o Lotti suprime os efeitos,
preservando os estilos selecionados para mais tarde.

## Atribua um estilo a cada tipo de conclusão

<ManualScreenshot
  caseId="celebrations/styles"
  alt="Seletor de estilo de celebração do Lotti com Tarefas selecionado e as opções Sparks, Fireworks, Confetti, Brasas, Bolhas, Aleatório e Combine dois"
  caption="Tarefas, hábitos e itens da lista de verificação podem usar estilos diferentes. Selecione um cartão para visualizá-lo e aplicá-lo."
/>

Para cada tipo de conclusão, escolha **Sparks**, **Fireworks**, **Confetti**,
**Brasas** ou **Bolhas**. Escolha **Aleatório** para ter um único estilo novo a
cada conclusão, ou **Combine dois** para sobrepor dois estilos selecionados
aleatoriamente.

Use **Experimente** para visualizar a seleção atual sem alterar dados reais de
tarefas ou hábitos.

## Ajuste um estilo individual

Selecione a ação de personalizar em um estilo fixo para abrir seu playground em
tela cheia.

<ManualScreenshot
  caseId="celebrations/playground"
  alt="Playground de celebração do Lotti Sparks com visualização de tarefas ao vivo e controles agrupados de forma, movimento e aparência"
  caption="As alterações são salvas e aplicadas em todos os lugares imediatamente. Reproduza a amostra após cada ajuste."
/>

Os controles disponíveis dependem do efeito e são agrupados em **Forma**,
**Movimento** e **Aparência**. Ajuste os controles deslizantes e use a amostra
destacada ou **Repetir** para ver o resultado. **Redefinir para o padrão**
restaura o estilo integrado; a confirmação oferece **Desfazer** caso a
redefinição tenha sido acidental.

A personalização é global para aquele estilo. Ela não fica vinculada à tarefa
de exemplo mostrada no playground nem a uma tarefa real específica.
