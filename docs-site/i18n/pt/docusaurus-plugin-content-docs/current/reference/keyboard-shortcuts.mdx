# Trabalhe com atalhos de teclado

No desktop, o Lotti oferece um catálogo pesquisável de comandos globais e
contextuais. Abra **Configurações → Atalhos de teclado** para ver cada comando
com a combinação resolvida para a sua plataforma.

## Abra a ajuda de atalhos rapidamente

Pressione **F1** ou **Primary + ?** para abrir a referência de atalhos sem
sair do seu trabalho atual. **Primário** significa Comando no macOS e Controle
no Windows ou Linux.

Entre os comandos globais mais úteis estão:

| Ação | Atalho |
| --- | --- |
| Abrir a paleta de comandos | Primário + K |
| Criar uma entrada de texto | Primário + N |
| Criar uma tarefa | Primário + T |
| Abrir Tarefas, Daily OS, Projetos ou Hábitos | Primário + 1/2/3/4 |
| Abrir Painéis, Diário, Eventos ou Configurações | Primário + 5/6/7/8 |

Páginas individuais adicionam comandos contextuais, como salvar, atualizar,
pesquisar ou criar. A disponibilidade deles pode depender do que está em foco
no momento ou de haver alterações não salvas.

## Execute um comando sem sair do seu trabalho

Pressione **Primary + K** para abrir a paleta de comandos sobre a página
atual. Ela combina ações globais do aplicativo — criar entradas, abrir
tarefas, navegar e aplicar zoom — com comandos fornecidos pela página ativa.

Comece a digitar para restringir a lista por nome de ação, grupo ou combinação
de teclas. Use as setas para cima e para baixo para mover o destaque, **Enter**
para executar o comando e **Escape** para fechar a paleta. Como a paleta é
contextual, um comando como **Salvar** só aparece quando o editor em foco pode
salvar naquele momento.

## Pesquise por ação ou tecla

<ManualScreenshot
  caseId="keyboard-shortcuts/search"
  alt="Catálogo de atalhos de teclado Lotti filtrado com uma consulta de pesquisa"
  caption="Pesquise o catálogo em vez de memorizá-lo. Os resultados são atualizados conforme você digita."
/>

Use o campo de pesquisa quando souber a ação ou parte da combinação de teclas.
O catálogo dentro do aplicativo é a referência definitiva caso um atalho mude
em uma versão posterior do Lotti.
