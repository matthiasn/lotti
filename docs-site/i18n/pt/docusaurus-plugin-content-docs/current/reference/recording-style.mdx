# Escolha um estilo de gravação

Abra **Configurações → Estilo de gravação** para escolher a visualização
mostrada enquanto o Lotti grava áudio. Isso altera apenas a apresentação;
ambos os estilos representam o mesmo processo de gravação.

## Compare os dois estilos

- **Moderno** mostra um orbe de energia animado que reage ao nível atual.
- **Analógico** mostra um medidor VU tradicional com uma agulha em movimento.

Selecione um dos cartões para torná-lo o estilo ativo. Não há uma ação
separada de Salvar ou Continuar.

## Visualize com sua voz

Ative **Tente com sua voz** para alimentar a pré-visualização com o
microfone. Fale à distância que você pretende usar durante a gravação e
observe como o visual selecionado responde. Desative a opção quando terminar
de comparar os estilos.

A pré-visualização ajuda você a escolher uma linguagem visual; ela não é uma
ferramenta de calibração de ganho de entrada ou de qualidade de gravação.
Consulte [Gravar áudio](../plan-and-capture/recordings.mdx) para conhecer o
fluxo de trabalho real de gravação.
