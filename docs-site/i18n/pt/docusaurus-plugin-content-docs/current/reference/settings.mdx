# Encontre configurações

As configurações são agrupadas pela capacidade que controlam. O desktop usa
uma árvore persistente com a página selecionada ao lado; o celular usa uma
lista focada e abre uma página de configurações por vez.

Escolha **Manual** na última seção de Configurações para abrir este guia
no seu navegador padrão. No macOS, **Ajuda → Manual** fornece o mesmo link.
O guia segue o idioma do seu sistema por padrão; escolha um idioma diferente
em **Configurações → Configurações avançadas → Idioma** quando necessário.

Destinos comuns incluem:

- **[Aparência](./appearance.mdx)** para os modos de exibição claro, escuro e
  automático, além de paletas de tema independentes.
- **[Estilo de gravação](./recording-style.mdx)** para o orbe de energia ou o
  medidor analógico mostrado durante a gravação.
- **[Fala](./speech.mdx)** para a voz no dispositivo, o modelo e a velocidade
  usados para ler resumos de tarefas em voz alta.
- **[Atalhos de teclado](./keyboard-shortcuts.mdx)** para o catálogo
  pesquisável de comandos do desktop.
- **[Celebrações de conclusão](./completion-celebrations.mdx)** para
  animações, respostas táteis, estilos e seu playground ao vivo.
- **[Configurações avançadas](./advanced-settings.mdx)** para sinalizadores de
  recursos, registro, ações de manutenção, diagnósticos de integração e
  informações da compilação.
- **Categorias** para limites de planejamento e **Rótulos** para tags
  transversais.
- **Hábitos** e **Painéis** para estruturas de reflexão.
- **Configurações de IA** para fornecedores, modelos, perfis e habilidades.
- **Daily OS** para o perfil de inferência e a saudação do planejador.
- **Sincronização** para o estado de conexão e atividade.

O conjunto exato pode variar conforme as capacidades da plataforma e os
recursos habilitados.

## Gerenciar definições

Abra **Configurações → Definições** para gerenciar as estruturas que organizam
seus dados. As categorias definem limites de planejamento e privacidade, os
rótulos marcam tarefas entre categorias, os hábitos definem rotinas
recorrentes, os painéis organizam visualizações de reflexão e os mensuráveis
definem os números que você acompanha ao longo do tempo.
