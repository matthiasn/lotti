# Navegue pelas Novidades

As Novidades apresentam notas de lançamento dentro do Lotti, correspondentes à
versão que você está executando. Elas podem abrir automaticamente após uma
primeira inicialização elegível ou uma mudança de versão, e também estão
disponíveis na entrada **Novidades** em Configurações quando esse recurso está
habilitado.

As notas de versão são ordenadas da mais recente para a mais antiga. Cada
página combina uma imagem de destaque específica do lançamento, a versão exata
do aplicativo e um breve relato das mudanças visíveis para o usuário.

## Leia a versão atual

O emblema de versão identifica o lançamento cujas notas estão na tela. **NOVO**
marca o lançamento mais novo do conjunto atual; não significa que todos os
itens da página estejam inacabados ou sejam experimentais.

O corpo rola de forma independente quando o lançamento é maior que a altura
disponível. Os links usam o manipulador de links normal do Lotti, e as imagens
são pré-carregadas quando disponíveis, para que a navegação entre as versões
continue fluida.

Use os controles do rodapé para navegar:

- a seta para a direita abre a versão mais antiga seguinte;
- a seta para a esquerda retorna a uma versão mais recente;
- os marcadores de posição mostram onde a página atual está no conjunto
  carregado;
- **Skip** fecha um conjunto de vários lançamentos antes da última página e
  marca todo o conjunto carregado como visto; e
- **Concluído** aparece na página mais antiga e também marca todo o conjunto
  como visto.

Fechar o modal arrastando-o para fora ou tocando na área escurecida ao redor é
mais conservador: o Lotti marca apenas os lançamentos que você realmente
alcançou, até a página mais distante visualizada. Lançamentos que você não
alcançou podem, portanto, aparecer na próxima vez que as Novidades forem
abertas.

## Revisite notas que você já dispensou

Quando não resta nenhum lançamento não visto, abrir as Novidades mostra o
estado **Você está em dia**. Escolha **Ver lançamentos anteriores** para limpar
os marcadores locais de visto e navegar novamente pelo conjunto de versões
disponível. Isso altera apenas o histórico local das notas de lançamento; não
altera a versão instalada do aplicativo nem baixa outra compilação.

As Novidades nunca mostram notas de uma versão mais recente que a versão
instalada do Lotti. Se o conteúdo da versão não puder ser carregado, o
aplicativo recorre a um estado vazio em vez de apresentar notas da compilação
errada.

Para o número exato da compilação, a plataforma, os totais locais de entradas
e as contagens de tarefas, consulte
[Sobre Lotti](./advanced-settings.mdx#verifique-a-compilação-e-os-totais-locais).
