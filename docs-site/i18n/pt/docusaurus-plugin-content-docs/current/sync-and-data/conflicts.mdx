# Resolver conflitos de sincronização

Um conflito aparece quando dois dispositivos alteram a mesma entrada de forma
independente e nenhuma das versões é claramente mais recente. O Lotti preserva
as duas versões em vez de escolher uma silenciosamente.

Abra **Configurações → Sincronizar → Conflitos**. A lista separa itens não
resolvidos e resolvidos e identifica o tipo de entidade e o horário de cada
conflito.

## Compare as duas versões

Abra um conflito para ver todos os campos que diferem. As alterações de texto
incluem destaques no nível de palavra; os campos inalterados são resumidos
para que a página não esconda o quanto da entrada ainda coincide.

  caseId="sync/conflict-detail"

Os três caminhos de resolução são:

- **Use este dispositivo** mantém a versão local completa.
- **Usar da sincronização** mantém a versão recebida completa.
- **Combinar** permite escolher um lado inicial e depois selecionar o vencedor
  de cada campo mesclável de forma independente.

Manter um lado completo é apropriado quando o outro está simplesmente
desatualizado. **Combinar** é a escolha mais segura quando campos separados
contêm trabalho válido dos dois dispositivos; o Lotti o marca como recomendado
quando vários campos mescláveis diferem.

## Combine campos sem redigitar

Selecione **Combinar**, escolha a versão de partida e alterne campos
individuais para o outro lado. Os campos que não podem ser mesclados de forma
independente seguem a versão inicial selecionada.

<ManualScreenshot
  caseId="sync/conflict-combine"
  alt="Editor de conflitos Combinar do Lotti com controle de versão inicial e escolhas separadas para o título e o corpo da tarefa"
  caption="Comece com uma entrada estruturalmente completa e, em seguida, pegue o melhor título e corpo deste dispositivo ou da sincronização."
/>

Revise todos os campos listados antes de selecionar **Aplicar combinado**. A
entrada resolvida recebe um histórico de sincronização mesclado para que as
mesmas duas edições não entrem em conflito de novo imediatamente.

:::caution[Uma resolução altera o resultado compartilhado]

Manter um lado ou aplicar uma versão combinada grava o vencedor localmente e
permite que ele seja sincronizado com os outros dispositivos. Leia a
comparação completa antes de confirmar. Se um lado excluiu a entrada enquanto
o outro a editava, o Lotti mostra a escolha mais segura entre excluir e
editar, em vez de uma mesclagem de campos normal.

:::
