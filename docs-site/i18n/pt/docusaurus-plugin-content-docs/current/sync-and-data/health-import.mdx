# Importar dados de saúde

A importação de saúde está disponível no iOS e no Android. Abra
**Configurações → Avançado → Health Import**, escolha uma data de início e uma
de término e selecione a família de dados que você quer que o Lotti leia do
armazenamento de dados de saúde do telefone.

As importações disponíveis são:

- **Atividade** para totais de atividades;
- **Sono** para amostras de sono;
- **Frequência cardíaca** para observações de pulso;
- **Pressão Arterial** para leituras de pressão arterial;
- **Medida Corporal** para leituras de métricas corporais; e
- **Treino** para treinos gravados.

Cada botão importa apenas sua própria família de dados no intervalo de datas exibido.
A alteração do intervalo não inicia uma importação por si só.

Seu sistema operacional pode solicitar permissão para dados de saúde. O Lotti
só consegue ler os tipos que você concede. Se uma família de dados esperada
estiver faltando, verifique as permissões de saúde do aplicativo nas
configurações do sistema do telefone e tente essa importação novamente.

A importação de saúde não é exibida no desktop porque os armazenamentos de
saúde móveis subjacentes não estão disponíveis lá. As observações importadas
ainda podem ser visualizadas nos painéis relevantes do Lotti depois de
sincronizadas com outro dispositivo.
