# Adăugați înregistrări și context

Lotti folosește un singur meniu **Adăugați** pentru lucrurile pe care le capturați în timpul zilei. Deschideți-l din jurnal pentru o înregistrare independentă sau dintr-o sarcină pentru a păstra noua înregistrare atașată acelei sarcini.

## Alegeți înregistrarea potrivită materialului

| Acțiune | O folosiți pentru |
| --- | --- |
| **Listă de verificare** | Un set scurt de pași concreți în sarcina curentă |
| **Sarcină** | Un rezultat separat care are nevoie de propria stare și propriul context |
| **Înregistrare audio** | O notă vocală care poate fi redată sau transcrisă |
| **Cronometru** | O sesiune de lucru cronometrată, legată de contextul curent |
| **Înregistrare text** | Note, decizii, observații sau informații lipite |
| **Importați imagine** | O imagine existentă pe dispozitiv |
| **Captură de ecran** | O nouă captură de ecran desktop atașată ca înregistrare imagine |

Meniul este contextual. **Listă de verificare** este disponibilă când o sarcină poate deține elementele, iar acțiunile specifice dispozitivului, precum **Captură de ecran**, apar numai pe platformele care le acceptă.

## Păstrați contextul cu munca

Când deschideți **Adăugați** dintr-o sarcină, Lotti leagă noua înregistrare de acea sarcină. Astfel, înregistrările, notele, imaginile și timpul rămân împreună pe pagina de detalii a sarcinii. Folosiți o înregistrare de jurnal independentă atunci când materialul nu aparține încă unui anumit rezultat; o puteți lega mai târziu.

Consultați [Înregistrați audio](./recordings.mdx) pentru capturare și transcriere sau [Lucrați cu sarcini](../organize-and-reflect/tasks.mdx) pentru structura sarcinilor, coperți și legături între sarcini.
