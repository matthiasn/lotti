# Alegeți aspectul Lotti

Deschideți **Setări → Aspect** pentru a controla felul în care arată Lotti. Modul de afișare și cele două alegeri de paletă sunt independente, astfel încât puteți asocia o temă luminoasă de zi cu una complet diferită pentru noapte.

## Selectați un mod de afișare

Alegeți unul dintre cele trei moduri:

- **Întunecat** folosește întotdeauna tema întunecată selectată.
- **Automat** urmează setarea luminoasă sau întunecată curentă a sistemului de operare.
- **Luminos** folosește întotdeauna tema luminoasă selectată.

Modul Automat este util când dispozitivul își schimbă aspectul după un program. Lotti urmează schimbarea și păstrează paleta aleasă pentru fiecare parte.

## Alegeți paletele luminoasă și întunecată

1. Selectați **Temă luminoasă** sau **Temă întunecată**.
2. Alegeți o paletă denumită din foaie.
3. Repetați pentru celălalt aspect, dacă este necesar.

<ManualScreenshot caseId="theming/picker" alt="Selectorul de teme Lotti care enumeră paletele denumite disponibile" caption="Selectorul de teme folosește aceeași structură de setări mobilă sau desktop ca aplicația." />

Nu există buton Salvați. Alegerea modului de afișare și a paletelor se păstrează imediat ce le selectați. Comutatorul propriu de lumină și întuneric al manualului este separat: schimbă documentația și alege automat varianta corespunzătoare de captură de ecran, dar nu modifică setările Lotti.
