# Lucrați cu comenzi rapide de la tastatură

Lotti pe desktop oferă un catalog căutabil de comenzi globale și contextuale. Deschideți **Setări → Comenzi rapide de la tastatură** pentru a vedea fiecare comandă cu combinația adaptată platformei dvs.

<ManualScreenshot caseId="keyboard-shortcuts/catalog" alt="Setările de comenzi rapide Lotti, cu comenzi desktop grupate după scop și combinațiile lor de taste curente" caption="Principal înseamnă Command pe macOS și Control pe Windows sau Linux; catalogul redă simbolurile corecte pentru platforma curentă." />

## Deschideți rapid ajutorul pentru comenzi

Apăsați **F1** sau **Principal + ?** pentru a deschide referința fără a părăsi munca curentă. **Principal** înseamnă Command pe macOS și Control pe Windows sau Linux.

Comenzile globale frecvent utile includ:

| Acțiune | Comandă rapidă |
| --- | --- |
| Deschideți paleta de comenzi | Principal + K |
| Creați o înregistrare text | Principal + N |
| Creați o sarcină | Principal + T |
| Deschideți Sarcini, Daily OS, Proiecte sau Obiceiuri | Principal + 1 / 2 / 3 / 4 |
| Deschideți Tablouri de bord, Jurnal, Evenimente sau Setări | Principal + 5 / 6 / 7 / 8 |

Paginile individuale adaugă comenzi contextuale, precum salvare, reîmprospătare, căutare sau creare. Disponibilitatea lor poate depinde de ceea ce este focalizat sau de existența modificărilor nesalvate.

## Rulați o comandă fără să părăsiți munca

Apăsați **Principal + K** pentru a deschide paleta de comenzi peste pagina curentă. Ea combină acțiuni globale — crearea înregistrărilor, deschiderea sarcinilor, navigarea și zoomul — cu comenzile oferite de pagina activă.

<ManualScreenshot caseId="keyboard-shortcuts/palette" alt="Paleta de comenzi Lotti deschisă peste Comenzi rapide de la tastatură, cu acțiuni disponibile de creare, navigare, vizualizare și editare" caption="Apar numai comenzile disponibile în contextul curent. Perspectiva mobilă demonstrează aceeași paletă desktop într-o fereastră îngustă; comenzile de tastatură sunt o funcție desktop." />

Începeți să tastați pentru a restrânge lista după numele acțiunii, grup sau combinație de taste. Folosiți săgețile sus și jos pentru a muta evidențierea, **Enter** pentru rulare și **Escape** pentru a închide paleta. Pentru că este contextuală, o comandă precum **Salvați** apare numai când editorul focalizat poate salva acum.

## Căutați după acțiune sau tastă

<ManualScreenshot caseId="keyboard-shortcuts/search" alt="Catalogul de comenzi rapide Lotti filtrat cu o interogare de căutare" caption="Căutați în catalog în loc să memorați comenzile. Rezultatele se actualizează pe măsură ce tastați." />

Folosiți câmpul de căutare când cunoașteți acțiunea sau o parte din combinația ei de taste. Catalogul din aplicație este autoritatea dacă o combinație se schimbă într-o versiune ulterioară a Lotti.
