# Alegeți un stil de înregistrare

Deschideți **Setări → Stil de înregistrare** pentru a alege vizualizarea afișată cât timp Lotti înregistrează audio. Aceasta schimbă numai prezentarea; ambele stiluri reprezintă același proces de înregistrare.

<ManualScreenshot caseId="recording-style/preview" alt="Setările Stil de înregistrare Lotti care compară orbul de energie Modern cu vumetrul Analog" caption="Selectați un stil pentru a-l păstra imediat, apoi previzualizați răspunsul cu niveluri simulate sau cu microfonul dvs." />

## Comparați cele două stiluri

- **Modern** arată un orb de energie animat, care reacționează la nivelul curent.
- **Analog** arată un vumetru tradițional cu ac mobil.

Selectați oricare card pentru a-l face stil activ. Nu există o acțiune separată Salvați sau Continuați.

## Previzualizați cu vocea dvs.

Activați **Încercați cu vocea dvs.** pentru a alimenta previzualizarea de la microfon. Vorbiți de la distanța la care vă așteptați să înregistrați și urmăriți cum răspunde vizualul selectat. Dezactivați opțiunea când ați terminat comparația.

Previzualizarea vă ajută să alegeți un limbaj vizual; nu este un instrument de calibrare a amplificării intrării sau a calității înregistrării. Consultați [Înregistrați audio](../plan-and-capture/recordings.mdx) pentru fluxul efectiv de înregistrare.
