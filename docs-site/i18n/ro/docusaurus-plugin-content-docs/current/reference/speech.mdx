# Alegeți vocea de citire

Când citirea cu voce tare este disponibilă, deschideți **Setări → Vorbire**. Aceste controale se aplică rezumatelor AI ale sarcinilor; ele nu schimbă capturarea prin microfon și nici furnizorul AI care transcrie o înregistrare.

<ManualScreenshot caseId="speech/settings" alt="Setările Vorbire Lotti, cu Masculin 3 selectat, modelul Supertonic 3 pe dispozitiv și viteza de citire 1,25" caption="Configurația exemplu din Centrul de comandă folosește Masculin 3 la 1,25× pentru briefinguri Project Waddle concise." />

## Alegeți o voce

Folosiți comutatorul **Feminin / Masculin** pentru a parcurge cele două grupuri, apoi selectați una dintre cele cinci voci numerotate ale grupului. Schimbarea grupurilor modifică numai lista vizibilă; selectați o voce numerotată pentru a o activa.

<ManualScreenshot caseId="speech/voice-options" alt="Setările Vorbire Lotti, cu cele cinci opțiuni de voce feminină pe dispozitiv" caption="Sunt disponibile zece stiluri de voce incluse: Feminin 1–5 și Masculin 1–5." />

Vocea selectată este o preferință locală a dispozitivului. Astfel, fiecare dispozitiv poate folosi vocea potrivită pentru difuzoarele și mediul său de ascultare, fără a schimba alegerea de pe celelalte dispozitive.

## Descărcarea modelului și confidențialitatea

Lotti folosește în prezent **Supertonic 3** pentru citirea cu voce tare. Sinteza rulează pe dispozitiv, în loc să trimită rezumatul la un serviciu vocal cloud. Modelul este mare și se descarcă la prima utilizare, așadar acordați timp, lățime de bandă și spațiu de stocare pentru configurarea inițială.

## Setați viteza de citire

Alegeți o viteză de la **0,5×** la **2×**. Valoarea **1×** este ritmul natural; valorile mai mici încetinesc briefingul, iar cele mai mari îl scurtează. Viteza aleasă este de asemenea stocată numai pe dispozitivul curent.

Butonul de citire cu voce tare apare doar când funcția este activată, platforma acceptă motorul vocal și sarcina are un rezumat AI nevid.

Pentru intrare vorbită și rezultate de recunoaștere salvate, consultați [Înregistrați audio](../plan-and-capture/recordings.mdx).
