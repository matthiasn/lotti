# Răsfoiți Ce este nou

Ce este nou prezintă notele de lansare în Lotti, potrivite versiunii pe care o rulați. Se poate deschide automat după o primă lansare eligibilă sau schimbare de versiune și este disponibilă și din intrarea **Ce este nou** în Setări, când capabilitatea este activată.

Notele de lansare sunt ordonate cu cele mai noi primele. Fiecare pagină combină o imagine principală specifică lansării, versiunea exactă a aplicației și o relatare scurtă a schimbărilor vizibile utilizatorului.

## Citiți versiunea curentă

Insigna versiunii identifică lansarea ale cărei note sunt pe ecran. **NOU** marchează cea mai recentă versiune din setul curent; nu înseamnă că fiecare element de pe pagină este neterminat sau experimental.

Corpul derulează independent când lansarea este mai lungă decât înălțimea disponibilă. Legăturile folosesc gestionarul obișnuit de legături Lotti, iar imaginile sunt încărcate înaintea schimbării paginii când sunt disponibile, pentru ca trecerea între lansări să rămână fluidă.

## Parcurgeți istoricul lansărilor

Folosiți controalele din subsol pentru navigare:

- săgeata spre dreapta deschide următoarea lansare mai veche;
- săgeata spre stânga revine la o lansare mai nouă;
- marcatorii de poziție arată unde se află pagina curentă în setul încărcat;
- **Omiteți** închide un set cu mai multe lansări înainte de ultima pagină și marchează întreg setul încărcat ca văzut; iar
- **Gata** apare pe cea mai veche pagină și marchează de asemenea întreg setul ca văzut.

Închiderea ferestrei prin tragere sau prin atingerea barierei din jur este mai conservatoare: Lotti marchează doar lansările la care ați ajuns, până la pagina cea mai îndepărtată văzută. Lansările neatinse pot apărea astfel data viitoare când se deschide Ce este nou.

## Revedeți note deja închise

Când nu mai rămâne nicio lansare nevăzută, deschiderea Ce este nou arată starea **Sunteți la curent**. Alegeți **Vedeți lansările anterioare** pentru a șterge marcajele locale de văzut și a răsfoi din nou setul disponibil. Aceasta schimbă numai istoricul local al notelor de lansare; nu modifică versiunea instalată și nu descarcă altă versiune.

Ce este nou nu arată niciodată note pentru o lansare mai nouă decât versiunea Lotti instalată. Dacă nu se poate încărca conținutul unei lansări, aplicația revine la o stare goală, în loc să prezinte note pentru versiunea greșită.

Pentru numărul exact al versiunii, platformă, totaluri locale de intrări și număr de sarcini, consultați [Despre Lotti](./advanced-settings.mdx#verificați-versiunea-și-totalurile-locale).
