# Importați date de sănătate

Importul de sănătate este disponibil pe iOS și Android. Deschideți **Setări → Avansat → Import sănătate**, alegeți o dată de început și una de sfârșit, apoi selectați familia de date pe care doriți ca Lotti să o citească din stocarea de date de sănătate a telefonului.

Importurile disponibile sunt:

- **Activitate** pentru totaluri de activitate;
- **Somn** pentru eșantioane de somn;
- **Puls** pentru observații de puls;
- **Tensiune arterială** pentru citiri de tensiune;
- **Măsurători corporale** pentru măsurători precum metricile corpului; și
- **Antrenament** pentru antrenamente înregistrate.

Fiecare buton importă numai propria familie de date în intervalul afișat. Schimbarea intervalului nu pornește singură un import.

Sistemul de operare poate cere permisiunea pentru datele de sănătate. Lotti poate citi doar tipurile pe care le permiteți. Dacă lipsește o familie de date așteptată, verificați permisiunile de sănătate ale aplicației în setările de sistem ale telefonului și încercați din nou importul.

Importul de sănătate nu este afișat pe desktop, deoarece stocările mobile de sănătate nu sunt disponibile acolo. Observațiile importate pot fi totuși văzute în tablourile de bord Lotti relevante după ce s-au sincronizat pe alt dispozitiv.
