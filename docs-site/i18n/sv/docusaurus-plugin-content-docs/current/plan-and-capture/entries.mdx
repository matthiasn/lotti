# Lägg till poster och kontext

Lotti använder en **Lägg till**-meny för det du fångar under dagen. Öppna den
från dagboken för en fristående post, eller från en uppgift för att behålla den nya posten
fäst vid den uppgiften.

## Välj det bidrag som matchar materialet

| Åtgärd | Använd den för |
| --- | --- |
| **Checklista** | En kort uppsättning konkreta steg i den aktuella uppgiften |
| **Uppgift** | Ett separat utfall som behöver sin egen status och kontext |
| **Ljudinspelning** | En röstnotis som kan spelas upp eller transkriberas |
| **Timer** | En tidsbegränsad arbetssession kopplad till den aktuella kontexten |
| **Textpost** | Anteckningar, beslut, observationer eller inklispad information |
| **Importera bild** | En befintlig bild från enheten |
| **Skärmdump** | En ny skrivbordsskärmdump bifogad som bildpost |

Menyn är kontextuell. **Checklista** finns tillgänglig när en uppgift kan äga
föremål, medan enhetsspecifika handlingar som **Skärmdump** bara visas där
plattformen stöder dem.

## Behåll kontexten kring arbetet

När du öppnar **Lägg till** från en uppgift länkar Lotti den nya posten tillbaka till den uppgiften.
Det håller ihop inspelningar, anteckningar, bilder och tidsregistreringar under uppgiften
detaljsidan. Använd en fristående dagbokspost när materialet inte hör hemma
till ett specifikt resultat än; Den kan länkas senare.

See [Record audio](./recordings.mdx) for capture and transcription, or
[Work with tasks](../organize-and-reflect/tasks.mdx) for task structure, cover
konst och uppdragslänkar.
