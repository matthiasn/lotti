# Välj Lottis utseende

Öppna **Inställningar → Tematisering** för att styra hur Lotti ser ut. Displayläget och
De två palettvalen är oberoende av varandra, så du kan kombinera en ljus dag
tema med ett helt annat natttema.

## Välj ett visningsläge

Välj ett av tre lägen:

- **Mörkt** använder alltid det valda mörka temat.
- ** Automatisk** följer operativsystemets nuvarande ljus- eller mörkläge.
- **Light** använder alltid det valda ljustemat.

Automatiskt läge är användbart när din enhet ändrar utseende enligt ett schema.
Lotti följer den förändringen samtidigt som hon behåller din valda palett för varje sida.

## Välj ljusa och mörka paletter

1. Välj **Ljus tema** eller **Mörk tema**.
2. Välj en namngiven palett från arket.
3. Upprepa för det andra utseendet om det behövs.

Det finns ingen Spara-knapp. Visningsläge och palettval finns kvar så snart som
Du väljer dem. Manualens egen ljus- och mörkerbrytare är separat: den ändras
dokumentationen och väljer automatiskt den matchande skärmdumpsvarianten,
men det ändrar inte dina Lotti-inställningar.
