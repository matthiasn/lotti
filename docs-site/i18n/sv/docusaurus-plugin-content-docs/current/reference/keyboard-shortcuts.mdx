# Arbeta med tangentbordsgenvägar

Desktop Lotti visar en sökbar katalog av app-omfattande och kontextuella
befallningar. Öppna **Inställningar → Tangentbordsgenvägar** för att se varje kommando med
Bindningen löst för din plattform.

## Öppna genvägar hjälper snabbt

Tryck **F1** eller **Primär + ?** för att öppna genvägsreferensen utan att lämna
Ditt nuvarande arbete. **Primär** betyder Command på macOS och Control på Windows eller
Linux.

Ofta användbara app-övergripande kommandon inkluderar:

| Handling | Genväg |
| --- | --- |
| Öppna kommandopaletten | Primär + K |
| Skapa en textpost | Primär + N |
| Skapa en uppgift | Primär + T |
| Öppna uppgifter, dagliga operativsystem, projekt eller vanor | Primär + 1 / 2 / 3 / 4 |
| Öppna instrumentpaneler, dagbok, evenemang eller inställningar | Primär + 5 / 6 / 7 / 8 |

Enskilda sidor lägger till kontextuella kommandon som spara, uppdatera, söka eller
Skapa. Deras tillgänglighet kan bero på vad som för närvarande är fokuserat eller om
Det finns osparade ändringar.

## Kör ett kommando utan att lämna ditt arbete

Tryck **Primär + K** för att öppna kommandopaletten över den aktuella sidan. Det
kombinerar app-omfattande åtgärder – skapa poster, öppna uppgifter, navigera och
Zooming—med kommandon från den aktiva sidan.

<ManualScreenshot
  caseId="keyboard-shortcuts/palette"

Börja skriva för att begränsa listan efter åtgärdsnamn, grupp eller tangentkombination. Användning
upp- och nedpilarna för att flytta markeringen, **Enter** för att köra den, och
**Escape** för att stänga paletten. Eftersom paletten är kontextuell, är ett kommando
som att **Spara** bara visas när den fokuserade redigeraren för närvarande kan spara.

Använd sökfältet när du vet antingen åtgärden eller en del av dess nyckel
kombination. Katalogen i appen är auktoriteten om en bindning ändras i en
senare version av Lotti.
