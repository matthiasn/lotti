# Välj en inspelningsstil

Öppna **Inställningar → Inspelningsstil** för att välja visualiseringen som visas medan
Lotti spelar in ljud. Detta ändrar endast presentationen; Båda stilarna representerar
Samma inspelningsprocess.

## Jämför de två stilarna

- **Modern** visar en animerad energikula som reagerar på den aktuella nivån.
- **Analogt** visar en traditionell VU-mätare med en rörlig nål.

Välj vilket kort som helst för att göra det till den aktiva stilen. Det finns ingen separat sparfil eller
Fortsätt agera.

## Förhandsvisning med din röst

Aktivera **Försök med din röst** för att styra förhandsvisningen från mikrofonen. Tala
på det avstånd du förväntar dig att använda när du spelar in och se hur de valda
Visuell svarar. Stäng av alternativet när du har jämfört stilar.

Förhandsvisningen hjälper dig att välja ett visuellt språk; det är inte en input-gain eller
recording-quality calibration tool. See [Record audio](../plan-and-capture/recordings.mdx)
för själva inspelningsarbetsflödet.
