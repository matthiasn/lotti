# Hitta inställningar

Inställningarna är grupperade efter den kapacitet de kontrollerar. Skrivbordet använder en persistent
träd med den valda sidan bredvid; Mobilen använder en fokuserad lista och öppnar en sådan
Inställningssida i taget.

Välj **Manual** i den sista delen av Inställningar för att öppna denna guide
i din standardwebbläsare. På macOS ger **Hjälp → manual** samma länk.
Guiden följer ditt systemspråk som standard; Välj ett annat språk
i **Inställningar → Avancerade inställningar → Språk** när det behövs.

Vanliga destinationer inkluderar:

- **[Appearance](./appearance.mdx)** for light, dark, and automatic display
  Fler oberoende temapaletter.
- **[Recording Style](./recording-style.mdx)** for the energy-orb or analogue
  Mätaren visas under inspelningen.
- **[Speech](./speech.mdx)** for the on-device voice, model, and speed used to
  Läs uppgiftssammanfattningar högt.
- **[Keyboard shortcuts](./keyboard-shortcuts.mdx)** for the searchable desktop
  Kommandokatalog.
- **[Completion celebrations](./completion-celebrations.mdx)** for animations,
  haptik, stilar och deras levande lekplats.
- **[Advanced Settings](./advanced-settings.mdx)** for feature flags, logging,
  underhållsåtgärder, onboarding-diagnostik och bygginformation.
- **Kategorier** för planeringsgränser och **Etiketter** för korsklippande taggar.
- **Habits** och **Dashboards** för reflektionsstrukturer.
- **AI-inställningar** för vårdgivare, modeller, profiler och färdigheter.
- **Daily OS** för planerarens slutsatsprofil och hälsning.
- ** Synkronisera för anslutning och aktivitetstillstånd.

Den exakta uppsättningen kan variera beroende på plattformens kapacitet och aktiverade funktioner.

## Hantera definitioner

Öppna **Inställningar → Definitioner** för att hantera de strukturer som organiserar dina
Data. Kategorier sätter planerings- och integritetsgränser, etiketter taggar uppgifter över
Kategorier, vanor definierar återkommande rutiner, instrumentpaneler ordnar reflektion
Visningar och mätbara faktorer definierar de siffror du följer över tid.
