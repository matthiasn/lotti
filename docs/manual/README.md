# Lotti Manual — Getting Started

Short, focused walkthroughs for the most common first‑time setups. Pick the one that matches what you want to do today; come back for the others when you need them.

| Walkthrough | What it covers | When to do it |
|---|---|---|
| [AI setup](../../GETTING_STARTED.md) | Wire up a model — local Ollama (Qwen 3.6, Voxtral) or cloud Gemini Flash. | First. Nothing in the agentic layer works without a configured provider. |
| [Your first task with cover art](getting-started-task-with-image.md) | Create a task, dictate into it, and generate cover art. The basic everyday workflow in one pass. | After AI setup. This is the loop you'll repeat every day. |
| [Sync between your devices](getting-started-sync.md) | Set up end‑to‑end encrypted Matrix sync using the `matrix_provisioner` CLI. | Once you have more than one device you want to keep in sync — phone + desktop, work laptop + home laptop, etc. |

Once you've finished these, the longer reference [Manual](../MANUAL.md) covers categories, habits, dashboards, measurables, and the rest of the logbook surface in detail.
