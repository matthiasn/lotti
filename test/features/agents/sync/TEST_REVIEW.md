## Generative (Glados) testing opportunities

- [x] **[MED]** `mergeAgentStateCounters` in `lib/features/agents/sync/agent_concurrent_resolver.dart:82` — Has example-based tests in `agent_concurrent_resolver_test.dart` (lines ~250–355) but no Glados property test. Key invariants for a Glados property:
  1. `result.wakeCounter.value >= local.wakeCounter.value` (CRDT join is ≥ each input) for every host.
  2. `result.wakeCounter.value >= incoming.wakeCounter.value` for every host.
  3. `mergeAgentStateCounters(winner=local, local, incoming) != mergeAgentStateCounters(winner=incoming, local, incoming)` only in the non-counter fields (winner's non-counter fields dominate, but counter join is the same).
  4. Idempotent: merging with itself is a no-op on counters.
  Generator: `GCounter` generator already defined in `agent_concurrent_resolver_test.dart:65`; combine two `GCounter` values for `local.wakeCounter` and `incoming.wakeCounter`.
  **RESOLVED (partially stale):** an element-wise-join + winner-independence Glados property already existed. Added the missing invariants as a second property: the join dominates both inputs per host (for `wakeCounter` AND `slots.totalSessionsCompleted`), and self-merge is a counter no-op (CRDT idempotence).

## Summary

- **1 oversized test file**: `agent_sync_service_test.dart` (2010 lines) — split into 3–4 logical concern files.
- **1 oversized impl file**: `agent_sync_service.dart` (641 lines) — extractable to `agent_message_chain.dart` + `agent_sync_transaction.dart`.

- **Biggest coverage gap**: `_recordAgentEntitySequence`/`_recordAgentLinkSequence` error-swallow paths and the `runInTransaction` deferred-rethrow path — these are the silent-failure branches on the sync hot path that are hardest to debug in production.
- **Biggest single opportunity**: adding the `maybeCompact` content-addressed dedup test (ADR 0017 convergence invariant) and the `captureWakeInputs` retraction path test, as both are core correctness properties of the new compaction/capture system with zero test coverage today.
